# Diseño UI — v1.64.0 Declaraciones de impuestos (DDJJ)

Referencia visual alineada a `DESIGN_SYSTEM.md`: Tailwind 3, Bootstrap Icons, layouts admin existentes.

## Pantallas

### Catálogo de impuestos (`/taxes`)

- Listado tipo tabla admin: nombre, jurisdicción, periodicidad, cuenta pasivo (código + nombre).
- Botón primario indigo **Nuevo impuesto**; acciones Editar / Eliminar por fila.
- Formularios create/edit: campos nombre, jurisdicción, select cuenta **pasivo** (`AccountingAccount` tipo pasivo), periodicidad (mensual / trimestral / anual), checkbox activo.

### Declaraciones juradas (`/tax-returns`)

- **Índice:** filtros impuesto, año, estado; tabla impuesto, período legible, declarado, aplicado, badge estado (borrador / confirmada / anulada).
- **Alta (wizard):** selector impuesto; año; mes condicionado a frecuencia (trimestre: mes inicio 1/4/7/10); monto declarado; notas; botón **Cargar anticipos disponibles** (AJAX JSON); tabla con checkbox incluir + monto aplicado editable; totales en tiempo real vía Alpine; submit **Guardar borrador**.
- **Show:** tarjetas montos (declarado, aplicado, diferencia); estado; link al Libro Diario si hay permiso contabilidad; tabla líneas de aplicación (FC/NC); acciones borrador: editar, confirmar, eliminar; confirmada: anular.

### Percepciones (`/purchase-perceptions`)

- Columna **Impuesto (DDJJ)** y selector en modal crear/editar asociando `tax_id`.

### Saldos (`/purchase-perceptions/balances`)

- Columnas **Imputado (DDJJ)** y **Disponible** además de anticipos cargados y saldo cuenta.

## UX

- Mensajes flash verde/rojo según patrones existentes.
- Estados DDJJ con badges (amarillo borrador, verde confirmada, gris anulada).
- Confirmaciones JS en confirmar / anular / eliminar borrador.
