## Advertencias / edge cases

- Excel: fila 1 = headers, datos desde fila **2** (a diferencia de laborales que salta fila de título).
- Columna C se llama "NIVEL" en el Excel pero contiene valores NBU (ej. Albumina código 15 → 1.5).
- Códigos pueden existir en catálogo de tests; el seeder hace upsert por `code`, no duplica tests.
- Es variante del 2016 genérico (~6 prácticas más); no confundir con `docs/2016.xlsx` / "Nomenclador 2016".
