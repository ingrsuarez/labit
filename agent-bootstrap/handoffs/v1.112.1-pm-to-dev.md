## Contexto del pedido

Tras **v1.112.0**, el motor de aprendizaje ya registra navegación del portal, pero
solo `/dashboard` muestra cards. Empleados con rol **solo portal** nunca llegan ahí.

El usuario pidió **cards + perfil**: accesos aprendidos arriba, bloque «Mi Perfil»
actual abajo (sin pestañas, sin eliminar el perfil).
