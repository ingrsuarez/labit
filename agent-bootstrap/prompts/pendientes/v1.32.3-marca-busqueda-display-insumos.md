# v1.32.3 — Marca en búsqueda y UX mejorada del buscador de insumos

**VERSIÓN:** v1.32.3  
**SLUG:** marca-busqueda-display-insumos  
**DEPENDENCIAS:** v1.32.0 (buscador facturas de compra), v1.32.2 (buscador remitos)  
**ESTIMACIÓN:** 1h  
**PRIORIDAD:** alta

---

## Descripción

Mejora el buscador de insumos en facturas de compra y remitos:

1. **Marca en búsqueda y display**: el endpoint incluye `brand` en el criterio de búsqueda
   y en los resultados. El dropdown muestra `Código — Nombre — Marca`.

2. **UX del ítem seleccionado**:
   - Cuando hay un insumo vinculado: el campo de texto desaparece y se muestra **solo el badge**
     `Código — Nombre — Marca` con un botón `×` para desvincular
   - Cuando no hay insumo vinculado: se muestra el campo de descripción libre (texto)
   - Al presionar Tab desde el campo de búsqueda/descripción → foco va a **Cantidad**
   - Al presionar Tab desde Cantidad → **Lote** → **Vencimiento** → **Notas**
   - Al presionar Enter en el campo de búsqueda (con el dropdown cerrado) → **agrega una nueva fila de ítem**

## PASO 1 — Actualizar el endpoint de búsqueda

En `SupplyController::search()`, agregar `brand` al criterio y a los campos retornados:

```php
public function search(Request $request)
{
    $q = $request->get('q', '');
    if (strlen($q) < 2) {
        return response()->json([]);
    }

    $supplies = Supply::active()
        ->where(function ($query) use ($q) {
            $query->where('name', 'like', "%{$q}%")
                  ->orWhere('code', 'like', "%{$q}%")
                  ->orWhere('brand', 'like', "%{$q}%");
        })
        ->orderBy('name')
        ->limit(15)
        ->get(['id', 'code', 'name', 'brand', 'tracks_lot']);

    return response()->json($supplies);
}
```

**Verificación:** `GET /supplies/search?q=labix` retorna insumos con esa marca.

## PASO 2 — Nueva función `supplySearch()` para facturas de compra

En `resources/views/purchase-invoices/create.blade.php`, reemplazar la función
`supplySearch()` existente con esta versión mejorada:

```javascript
function supplySearch(item, index) {
    return {
        searchText: '',
        results: [],
        showResults: false,
        loading: false,
        highlightedIndex: -1,

        // Helper: construye el label completo del insumo
        buildLabel(supply) {
            return supply.brand
                ? `${supply.name} — ${supply.brand}`
                : supply.name;
        },

        async doSearch() {
            if (this.searchText.length < 2) {
                this.results = [];
                this.showResults = false;
                return;
            }
            this.loading = true;
            try {
                const resp = await fetch(`{{ route('supplies.search') }}?q=${encodeURIComponent(this.searchText)}`);
                this.results = await resp.json();
                this.showResults = true;
                this.highlightedIndex = -1;
            } catch (e) {
                this.results = [];
            } finally {
                this.loading = false;
            }
        },

        selectSupply(supply) {
            const label = this.buildLabel(supply);
            item.supply_id     = supply.id;
            item.description   = label;
            item._supply_code  = supply.code;
            item._supply_label = label;
            item._tracks_lot   = supply.tracks_lot;
            this.searchText    = '';
            this.showResults   = false;
            // Foco a Cantidad
            this.$nextTick(() => {
                document.querySelector(`#item-qty-${index}`)?.focus();
            });
        },

        unlinkSupply() {
            item.supply_id     = '';
            item._supply_code  = '';
            item._supply_label = '';
            item.description   = '';
            this.$nextTick(() => {
                this.$el.querySelector('input[type="text"]')?.focus();
            });
        },

        highlightNext() {
            if (this.highlightedIndex < this.results.length - 1) this.highlightedIndex++;
        },

        highlightPrev() {
            if (this.highlightedIndex > 0) this.highlightedIndex--;
        },

        selectHighlighted() {
            if (this.highlightedIndex >= 0 && this.highlightedIndex < this.results.length) {
                this.selectSupply(this.results[this.highlightedIndex]);
            }
        },

        // Tab: si dropdown abierto con highlight → seleccionar; si no → cerrar y pasar a Cantidad
        onTab(event) {
            if (this.showResults && this.highlightedIndex >= 0) {
                event.preventDefault();
                this.selectHighlighted();
            } else {
                this.showResults = false;
                // Guardar descripción libre si no hay insumo vinculado
                if (!item.supply_id) {
                    item.description = this.searchText;
                }
                // El Tab nativo llevará al siguiente campo (Cantidad por orden DOM)
            }
        },

        // Enter: si dropdown abierto → seleccionar highlight o primer resultado
        //        si dropdown cerrado → agregar nueva fila
        onEnter(event) {
            event.preventDefault();
            if (this.showResults && this.results.length > 0) {
                const idx = this.highlightedIndex >= 0 ? this.highlightedIndex : 0;
                this.selectSupply(this.results[idx]);
            } else {
                this.showResults = false;
                if (!item.supply_id) item.description = this.searchText;
                // Disparar evento para que invoiceForm() agregue una nueva fila
                this.$dispatch('add-item');
            }
        },

        init() {
            // Pre-llenar si el item ya tiene insumo (ej: edición)
            if (item.supply_id && item._supply_label) {
                // no tocar searchText: el badge se muestra en lugar del input
            } else if (item.description) {
                this.searchText = item.description;
            }
        }
    }
}
```

## PASO 3 — Template de la celda de descripción/insumo en facturas de compra

Reemplazar el `<td>` de la columna "Descripción / Insumo" con esta estructura:

```html
<td class="px-3 py-2 relative" x-data="supplySearch(item, index)"
    @add-item.window="/* ignorar eventos de otras filas */">

    <input type="hidden" :name="'items[' + index + '][supply_id]'"    :value="item.supply_id || ''">
    <input type="hidden" :name="'items[' + index + '][description]'"   :value="item.description || ''">

    {{-- Estado SIN insumo vinculado: mostrar campo de búsqueda/descripción libre --}}
    <template x-if="!item.supply_id">
        <div>
            <input type="text"
                   x-model="searchText"
                   @input.debounce.300ms="doSearch()"
                   @focus="if (searchText.length >= 2) showResults = true"
                   @keydown.arrow-down.prevent="highlightNext()"
                   @keydown.arrow-up.prevent="highlightPrev()"
                   @keydown.enter="onEnter($event)"
                   @keydown.escape="showResults = false; results = []"
                   @keydown.tab="onTab($event)"
                   placeholder="Buscar insumo o descripción libre..."
                   :id="'item-desc-' + index"
                   class="w-full rounded border-gray-300 text-sm focus:border-zinc-500 focus:ring-zinc-500">

            {{-- Dropdown de resultados --}}
            <div x-show="showResults && results.length > 0" x-cloak
                 @click.outside="showResults = false"
                 class="absolute z-50 left-0 right-0 mt-1 bg-white border border-gray-200 rounded-lg shadow-lg max-h-60 overflow-y-auto">

                <template x-for="(supply, i) in results" :key="supply.id">
                    <button type="button"
                            @click="selectSupply(supply)"
                            @mouseenter="highlightedIndex = i"
                            :class="highlightedIndex === i ? 'bg-teal-50 text-teal-800' : 'text-gray-700'"
                            class="w-full text-left px-3 py-2 text-sm hover:bg-gray-50 flex items-center gap-2 border-b border-gray-100 last:border-0">
                        <span class="font-mono text-xs text-gray-400 shrink-0" x-text="supply.code"></span>
                        <span class="flex-1" x-text="supply.name"></span>
                        <template x-if="supply.brand">
                            <span class="text-xs text-gray-400 shrink-0" x-text="supply.brand"></span>
                        </template>
                    </button>
                </template>

                {{-- Sin resultados --}}
                <div x-show="results.length === 0 && searchText.length >= 2 && !loading"
                     class="px-3 py-2 text-sm text-gray-400">
                    No se encontraron insumos. Escribí una descripción libre o presioná Enter para agregar ítem.
                </div>
            </div>
        </div>
    </template>

    {{-- Estado CON insumo vinculado: mostrar solo el badge --}}
    <template x-if="item.supply_id">
        <div class="flex items-center gap-2 py-1">
            <span class="font-mono text-xs text-gray-400 shrink-0" x-text="item._supply_code"></span>
            <span class="text-sm text-gray-800 flex-1" x-text="item._supply_label || item.description"></span>
            <button type="button"
                    @click="unlinkSupply()"
                    class="text-gray-300 hover:text-red-500 text-base leading-none shrink-0"
                    title="Desvincular insumo">×</button>
        </div>
    </template>

    {{-- Campos de lote/vencimiento si el insumo controla lote --}}
    {{-- (se muestran en las columnas dedicadas de la tabla, no aquí) --}}
</td>
```

## PASO 4 — Tab flow: IDs en los campos de la fila

Para que Tab funcione correctamente entre los campos de cada ítem, asegurarse de que
todos los inputs de una fila tienen el mismo `tabindex` o están en orden DOM correcto.

Agregar `:id` a los campos de Cantidad, Lote, Vencimiento y Notas en cada fila:

```html
{{-- Cantidad --}}
<input type="number" x-model.number="item.quantity"
       :id="'item-qty-' + index"
       class="...">

{{-- Lote --}}
<input type="text" x-model="item.lot_number"
       :id="'item-lot-' + index"
       placeholder="Lote"
       class="...">

{{-- Vencimiento --}}
<input type="date" x-model="item.expiration_date"
       :id="'item-exp-' + index"
       class="...">

{{-- Notas --}}
<input type="text" x-model="item.notes"
       :id="'item-notes-' + index"
       placeholder="—"
       class="...">
```

Tab order natural (izquierda a derecha en el DOM): Descripción/Insumo → Cantidad → Lote → Vencimiento → Notas → (siguiente fila).
