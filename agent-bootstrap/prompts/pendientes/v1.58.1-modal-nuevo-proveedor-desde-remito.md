## Descripción

Mismo botón "+" y modal de Nuevo Proveedor ya implementado en v1.58.0 para FC,
ahora aplicado a los formularios de **crear y editar Remito** (`delivery-notes/create`
y `delivery-notes/edit`).

El endpoint `POST /suppliers/store-inline` ya existe desde v1.58.0. Solo hay que
agregar el botón, el modal (copiado del create de FC) y el estado/métodos Alpine.js
adaptados a los nombres de variables del formulario de remito.

Diferencia clave respecto a FC:
- Variable Alpine del select: `supplierIdForDup` (en lugar de `supplier_id`)
- Evento al cambiar proveedor: `checkDuplicateRemito()` (en lugar de `onSupplierChange()`)
- No hay recarga de remitos disponibles (no aplica en este contexto)

### 1.3 — Estado y métodos Alpine.js

En el objeto `x-data` del formulario de remito, agregar las propiedades:

```js
// --- Modal nuevo proveedor ---
showNewSupplierModal: false,
inlineSuppliers: [],
newSupplierErrors: {},
newSupplierLoading: false,
newSupplierForm: {
    name: '',
    business_name: '',
    tax_id: '',
    tax_condition: '',
    email: '',
    phone: '',
    address: '',
    city: '',
    state: '',
    country: 'Argentina',
    postal: '',
    cbu: '',
    bank_alias: '',
    bank_name: '',
    contact_name: '',
    contact_phone: '',
    notes: '',
},
```

Y los métodos (notar que usa `supplierIdForDup` y `checkDuplicateRemito`):

```js
openNewSupplierModal() {
    this.newSupplierErrors = {};
    this.newSupplierForm = {
        name: '', business_name: '', tax_id: '', tax_condition: '',
        email: '', phone: '', address: '', city: '', state: '',
        country: 'Argentina', postal: '', cbu: '', bank_alias: '',
        bank_name: '', contact_name: '', contact_phone: '', notes: '',
    };
    this.showNewSupplierModal = true;
},

async submitNewSupplier() {
    this.newSupplierLoading = true;
    this.newSupplierErrors = {};
    try {
        const resp = await fetch('{{ route('suppliers.store-inline') }}', {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Accept': 'application/json',
                'X-CSRF-TOKEN': document.querySelector('meta[name="csrf-token"]').content,
            },
            body: JSON.stringify(this.newSupplierForm),
        });
        const data = await resp.json();
        if (!resp.ok) {
            this.newSupplierErrors = data.errors ?? { name: [data.message ?? 'Error al guardar.'] };
            return;
        }
        this.inlineSuppliers.push({ id: data.id, name: data.name });
        this.supplierIdForDup = String(data.id);
        this.showNewSupplierModal = false;
        await this.checkDuplicateRemito();
    } catch (e) {
        this.newSupplierErrors = { name: ['Error de red. Intentá nuevamente.'] };
    } finally {
        this.newSupplierLoading = false;
    }
},
```
