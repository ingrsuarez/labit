## Descripción

Implementar en LISCOM (Django) la primera pieza de la integración con labit:

- **Cliente HTTP** a la API v1 de labit con manejo de errores tipado y retries.
- **Modelos locales** en sqlite para cachear protocolos (clínico, muestras, vet) y
  sus determinaciones, indexados por barcode/protocol_number.
- **Servicio de sincronización** con tres modos: full del día, incremental, on-demand.
- **Comando management** `sync_protocols` para invocación manual o por cron.
- **UI básica** (Django templates + Tailwind):
  - Configuración de la conexión a labit
  - Listado de protocolos cacheados con buscador
  - Detalle de protocolo con `raw_data` para debugging
  - Historial de sync (últimos 100)
- **Singleton `LabitConnection`**: una sola configuración activa (topología
  "una instancia liscom por sede").
- **Sin cola de envío todavía** — eso es v1.52.0.
- **Sin recepción HL7 de resultados todavía** — eso es v1.50.0.
- **Sin mapeo de códigos por equipo todavía** — eso es v1.49.0.

Esta versión deja a LISCOM listo para que en v1.49.0 pueda responder a los equipos por HL7
sin necesidad de consultar a labit en tiempo real (excepto el caso fallback on-demand).

```python
"""Cliente HTTP para la API v1 de labit."""
from __future__ import annotations
import logging
from datetime import datetime, date
from typing import Optional
import requests
from requests.adapters import HTTPAdapter
from urllib3.util.retry import Retry

logger = logging.getLogger(__name__)


class LabitError(Exception):
    """Base de errores del cliente labit."""


class LabitConnectionError(LabitError):
    """No se pudo conectar (timeout, DNS, network)."""


class LabitAuthError(LabitError):
    """API key inválida o cliente desactivado (401)."""


class LabitNotFoundError(LabitError):
    """Recurso no encontrado (404)."""


class LabitServerError(LabitError):
    """Error 5xx después de retries."""


class LabitClient:
    """Cliente para la API v1 de labit.
    
    Configuración leída del singleton LabitConnection.
    """
    
    def __init__(self, base_url: str, api_key: str, timeout: int = 10):
        self.base_url = base_url.rstrip('/')
        self.api_key = api_key
        self.timeout = timeout
        self.session = self._build_session()
    
    @classmethod
    def from_active_connection(cls) -> 'LabitClient':
        from core.models import LabitConnection
        conn = LabitConnection.get_active()
        if not conn:
            raise LabitError('No hay LabitConnection activa configurada')
        return cls(conn.base_url, conn.api_key, conn.timeout_seconds)
    
    def _build_session(self) -> requests.Session:
        session = requests.Session()
        retry = Retry(
            total=3,
            backoff_factor=0.5,
            status_forcelist=[500, 502, 503, 504],
            allowed_methods=['GET'],
        )
        session.mount('http://', HTTPAdapter(max_retries=retry))
        session.mount('https://', HTTPAdapter(max_retries=retry))
        return session
    
    def _headers(self) -> dict:
        return {
            'X-API-Key': self.api_key,
            'Accept': 'application/json',
            'User-Agent': 'LISCOM/1.0',
        }
    
    def _get(self, path: str, params: Optional[dict] = None) -> dict:
        url = f"{self.base_url}{path}"
        try:
            r = self.session.get(url, headers=self._headers(),
                                  params=params or {}, timeout=self.timeout)
        except requests.exceptions.Timeout as e:
            raise LabitConnectionError(f"Timeout: {e}") from e
        except requests.exceptions.ConnectionError as e:
            raise LabitConnectionError(f"Connection error: {e}") from e
        
        if r.status_code == 401:
            raise LabitAuthError(f"Auth failed: {r.text}")
        if r.status_code == 404:
            raise LabitNotFoundError(f"Not found: {path}")
        if r.status_code >= 500:
            raise LabitServerError(f"Server error {r.status_code}: {r.text}")
        if r.status_code >= 400:
            raise LabitError(f"HTTP {r.status_code}: {r.text}")
        
        return r.json()
    
    def ping(self) -> dict:
        """Verifica conectividad y auth. Retorna {status, client, branch, company, time}."""
        return self._get('/api/v1/ping')
    
    def fetch_protocol_by_barcode(self, code: str) -> Optional[dict]:
        """Lookup directo por barcode. None si 404."""
        try:
            return self._get(f'/api/v1/protocols/by-barcode/{code}')
        except LabitNotFoundError:
            return None
    
    def fetch_protocols_today(self, page: int = 1, per_page: int = 100) -> dict:
        """Trae protocolos del día actual con paginación."""
        return self._get('/api/v1/protocols', params={
            'page': page, 'per_page': per_page,
        })
    
    def fetch_protocols_since(self, since: datetime,
                                page: int = 1, per_page: int = 100) -> dict:
        """Sync incremental: solo registros cambiados después de `since`."""
        return self._get('/api/v1/protocols', params={
            'updated_since': since.isoformat(),
            'page': page, 'per_page': per_page,
        })
    
    def fetch_protocols_range(self, date_from: date, date_to: date,
                                page: int = 1, per_page: int = 100) -> dict:
        """Búsqueda histórica en un rango de fechas."""
        return self._get('/api/v1/protocols', params={
            'date_from': date_from.isoformat(),
            'date_to': date_to.isoformat(),
            'page': page, 'per_page': per_page,
        })
```

```python
"""Servicio de sincronización de protocolos desde labit a la DB local."""
from __future__ import annotations
import logging
from datetime import datetime, date
from typing import Optional
from django.utils import timezone
from django.db import transaction
from core.models import LocalProtocol, LocalDetermination, SyncLog, LabitConnection
from .labit_client import LabitClient, LabitError

logger = logging.getLogger(__name__)


class ProtocolSyncService:
    
    def __init__(self, client: Optional[LabitClient] = None):
        self.client = client or LabitClient.from_active_connection()
    
    def sync_today(self, trigger: str = 'manual') -> SyncLog:
        log = SyncLog.objects.create(trigger=trigger, mode='today')
        try:
            result = self._sync_paginated(self.client.fetch_protocols_today)
            self._finalize_log(log, result, success=True)
        except Exception as e:
            self._finalize_log(log, None, success=False, error=str(e))
            logger.exception('sync_today failed')
        return log
    
    def sync_incremental(self, trigger: str = 'scheduled') -> SyncLog:
        conn = LabitConnection.get_active()
        since = conn.last_sync_at if conn and conn.last_sync_at else timezone.now() - timezone.timedelta(days=1)
        log = SyncLog.objects.create(trigger=trigger, mode='incremental',
                                       parameters={'since': since.isoformat()})
        try:
            result = self._sync_paginated(
                lambda page, per_page: self.client.fetch_protocols_since(since, page, per_page))
            self._finalize_log(log, result, success=True)
            if conn:
                conn.last_sync_at = log.started_at
                conn.save(update_fields=['last_sync_at'])
        except Exception as e:
            self._finalize_log(log, None, success=False, error=str(e))
            logger.exception('sync_incremental failed')
        return log
    
    def sync_range(self, date_from: date, date_to: date,
                    trigger: str = 'on_demand_range') -> SyncLog:
        log = SyncLog.objects.create(trigger=trigger, mode='range',
                                       parameters={'date_from': date_from.isoformat(),
                                                   'date_to': date_to.isoformat()})
        try:
            result = self._sync_paginated(
                lambda page, per_page: self.client.fetch_protocols_range(date_from, date_to, page, per_page))
            self._finalize_log(log, result, success=True)
        except Exception as e:
            self._finalize_log(log, None, success=False, error=str(e))
            logger.exception('sync_range failed')
        return log
    
    def sync_on_demand(self, barcode: str) -> Optional[LocalProtocol]:
        """Pull único cuando el scan no encuentra el barcode en cache."""
        log = SyncLog.objects.create(trigger='on_scan', mode='single',
                                       parameters={'barcode': barcode})
        try:
            data = self.client.fetch_protocol_by_barcode(barcode)
            if not data:
                self._finalize_log(log, {'added': 0, 'updated': 0}, success=True)
                return None
            
            protocol = self._upsert_protocol(data)
            self._finalize_log(log, {'added': 1, 'updated': 0}, success=True)
            return protocol
        except Exception as e:
            self._finalize_log(log, None, success=False, error=str(e))
            logger.exception('sync_on_demand failed for %s', barcode)
            return None
    
    def _sync_paginated(self, fetch_fn) -> dict:
        added, updated, failed = 0, 0, 0
        page = 1
        while True:
            response = fetch_fn(page=page, per_page=100)
            items = response.get('data', [])
            for item in items:
                try:
                    _, created = self._upsert_protocol_with_status(item)
                    if created:
                        added += 1
                    else:
                        updated += 1
                except Exception:
                    logger.exception('upsert failed for %s', item.get('protocol_number'))
                    failed += 1
            
            meta = response.get('meta', {})
            if page >= meta.get('last_page', 1):
                break
            page += 1
        
        return {'added': added, 'updated': updated, 'failed': failed}
    
    @transaction.atomic
    def _upsert_protocol_with_status(self, data: dict) -> tuple[LocalProtocol, bool]:
        defaults = {
            'barcode': data.get('barcode', data['protocol_number']),
            'type': data['type'],
            'remote_id': data['id'],
            'date': data['date'],
            'status': data['status'],
            'lab_branch_name': data.get('lab_branch', {}).get('name', ''),
            'patient_data': data.get('patient', {}),
            'raw_data': data,
            'updated_at_remote': data.get('updated_at'),
        }
        protocol, created = LocalProtocol.objects.update_or_create(
            protocol_number=data['protocol_number'], defaults=defaults)
        
        # Reemplazar determinaciones (estrategia simple para v1.48.0)
        protocol.determinations.all().delete()
        for det in data.get('determinations', []):
            LocalDetermination.objects.create(
                protocol=protocol,
                remote_id=det['id'],
                test_id=det.get('test_id'),
                test_code=det.get('test_code') or '',
                test_name=det.get('test_name', ''),
                material_name=det.get('material', {}).get('name', '') if det.get('material') else '',
                material_abbreviation=det.get('material', {}).get('abbreviation', '') if det.get('material') else '',
                unit=det.get('unit', ''),
                reference_value=det.get('reference_value', ''),
                status=det.get('status', ''),
                has_result=det.get('has_result', False),
            )
        return protocol, created
    
    def _upsert_protocol(self, data: dict) -> LocalProtocol:
        protocol, _ = self._upsert_protocol_with_status(data)
        return protocol
    
    def _finalize_log(self, log: SyncLog, result: Optional[dict],
                       success: bool, error: str = ''):
        log.finished_at = timezone.now()
        log.success = success
        log.error_message = error
        if result:
            log.protocols_added = result.get('added', 0)
            log.protocols_updated = result.get('updated', 0)
            log.protocols_failed = result.get('failed', 0)
        log.save()
```

```python
from datetime import datetime
from django.core.management.base import BaseCommand, CommandError
from integrations.sync_service import ProtocolSyncService


class Command(BaseCommand):
    help = 'Sincroniza protocolos desde labit a la DB local'
    
    def add_arguments(self, parser):
        parser.add_argument('--today', action='store_true',
                             help='Sync completa del día actual (default: incremental)')
        parser.add_argument('--barcode', type=str,
                             help='Sync único de un barcode específico')
        parser.add_argument('--range', nargs=2, metavar=('FROM', 'TO'),
                             help='Sync de rango YYYY-MM-DD YYYY-MM-DD')
    
    def handle(self, *args, **options):
        service = ProtocolSyncService()
        
        if options['barcode']:
            protocol = service.sync_on_demand(options['barcode'])
            if protocol:
                self.stdout.write(self.style.SUCCESS(f'Sincronizado: {protocol}'))
            else:
                self.stdout.write(self.style.WARNING(f'No encontrado: {options["barcode"]}'))
            return
        
        if options['range']:
            try:
                date_from = datetime.strptime(options['range'][0], '%Y-%m-%d').date()
                date_to = datetime.strptime(options['range'][1], '%Y-%m-%d').date()
            except ValueError as e:
                raise CommandError(f'Formato de fecha inválido: {e}')
            log = service.sync_range(date_from, date_to)
        elif options['today']:
            log = service.sync_today()
        else:
            log = service.sync_incremental()
        
        msg = (f'Sync {log.mode}: añadidos={log.protocols_added}, '
                f'actualizados={log.protocols_updated}, '
                f'fallados={log.protocols_failed}, '
                f'duración={log.duration_seconds:.1f}s')
        
        if log.success:
            self.stdout.write(self.style.SUCCESS(msg))
        else:
            self.stdout.write(self.style.ERROR(f'{msg} — Error: {log.error_message}'))
```

## Notas / Consideraciones

- **Singleton no enforced en BD:** la restricción de "una sola conexión activa" se hace
  en `clean()` del modelo. NO uses unique constraint en BD porque sqlite tiene rarezas
  con bool unique. Si se viola desde código, el primer `get_active()` devuelve el primero
  encontrado — aceptable para v1.48.0.

- **API key en sqlite plana:** simple por ahora. Si se quiere encriptar, sumar
  `cryptography` y usar `Fernet` con key derivada de `DJANGO_SECRET_KEY`. NO lo hagas en
  esta versión salvo que el usuario lo pida explícitamente.

- **Sin Celery/APScheduler:** la sync periódica se programa por cron / systemd timer.
  El comando management ya está pensado para ese uso. Si en futuro se quiere mover dentro
  del proceso ASGI, evaluar en v1.52.0 cuando se sume el outbox de envío.

- **Reemplazo de determinaciones (delete+create):** estrategia simple. Si una
  determinación tenía `has_result=True` localmente y la sincronizamos de nuevo, se pierde
  el flag local porque viene de labit. ESO ESTÁ BIEN — labit es la fuente de verdad para
  determinations metadata. Los resultados que liscom maneja viven en otra tabla
  (a crear en v1.50.0).

- **Bootstrap sistema agentes:** el usuario va a copiar `.agents/` y `agent-bootstrap/`
  desde labit a este repo. Si todavía no se hizo, hacer ese paso PRIMERO antes de empezar
  a implementar.

- **Eager loading en LocalProtocol list:** `prefetch_related('determinations')` en el
  controller `protocols_list` evita N+1.

- **Test environment:** sqlite en memoria para los tests via pytest-django default.
  No tocar la DB real en tests.

- **Designer:** la UI está pensada para operadores técnicos del laboratorio que ya usan
  el resto de LISCOM. Mantener consistencia visual con las pantallas existentes
  (ver `templates/` para el layout actual). El doc de diseño debe especificar la UX
  pero no inventar un sistema visual nuevo.

- **Performance esperado:** una sede con 200-500 protocolos/día, sync diaria <5s, sync
  incremental cada 30 min <1s. Si supera esto en producción, abrir hotfix.
