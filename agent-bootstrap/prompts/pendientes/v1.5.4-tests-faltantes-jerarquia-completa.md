## PASO 3 — Fix: eliminar padre cascadea a hijos

En `app/Http/Controllers/SampleController.php`, método `removeDetermination`:
al eliminar un padre, eliminar también sus hijos de la muestra.

**Antes:**
```php
public function removeDetermination(Sample $sample, SampleDetermination $determination)
{
    if ($determination->sample_id !== $sample->id) {
        abort(403);
    }
    $determination->delete();
    return back()->with('success', 'Determinación eliminada correctamente.');
}
```

**Después:**
```php
public function removeDetermination(Sample $sample, SampleDetermination $determination)
{
    if ($determination->sample_id !== $sample->id) {
        abort(403);
    }

    $test = $determination->test;
    $allChildren = $test->getAllChildren(false);
    $childTestIds = $allChildren->pluck('id');
    $deletedChildren = 0;

    if ($childTestIds->isNotEmpty()) {
        $deletedChildren = $sample->determinations()
            ->whereIn('test_id', $childTestIds)
            ->delete();
    }

    $determination->delete();

    $msg = 'Determinación eliminada correctamente.';
    if ($deletedChildren > 0) {
        $msg = "Determinación y {$deletedChildren} subdeterminaciones eliminadas correctamente.";
    }

    return back()->with('success', $msg);
}
```

**Verificación:**
1. Agregar Hemograma a un protocolo → se agregan 8 hijos
2. Eliminar Hemograma → se eliminan los 8 hijos también
3. Agregar un test standalone → eliminarlo solo elimina ese test
4. Agregar un hijo suelto (sin padre) → eliminarlo solo elimina ese test
