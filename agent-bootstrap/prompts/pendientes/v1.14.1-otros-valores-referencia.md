## Descripción

Las determinaciones tienen valores de referencia numéricos (Valor Mínimo / Valor Máximo)
que se muestran en los informes PDF. Sin embargo, muchas prácticas (especialmente de
aguas/alimentos) usan valores de referencia no numéricos como:
- "Ausencia en 100ml"
- "Positivo / Negativo"
- "< 10 UFC/ml"
- "Reactivo / No reactivo"
- "Apto / No apto"

Se agrega un campo `other_reference` en la tabla `tests` para almacenar estos valores.
Si tiene contenido, se muestra en la columna "Valores de ref." del informe PDF,
complementando o reemplazando los rangos numéricos min/max.

## PASO 5 — Incorporar en la construcción de reference_value

En `app/Http/Controllers/SampleController.php`, buscar el método `buildReferenceValue()`.

Si el test tiene `other_reference` y los campos `low`/`high` están vacíos, usar
`other_reference` como valor de referencia. Si ambos tienen contenido, concatenar.

Lógica sugerida dentro del método o donde se construye el valor de referencia:

```php
// Después de construir el reference_value basado en low/high:
if (empty($refValue) && !empty($test->other_reference)) {
    $refValue = $test->other_reference;
} elseif (!empty($test->other_reference)) {
    $refValue = $refValue . ' | ' . $test->other_reference;
}
```

Aplicar la misma lógica donde se construyan valores de referencia para admisiones
clínicas (`LabAdmissionController` si aplica).

## PASO 6 — Mostrar en PDFs

### 6.1 — PDF de muestras (`resources/views/sample/pdf-mpdf.blade.php`)

En las filas de determinaciones (hijos y standalone), donde se muestra
`$det->reference_value`, agregar fallback al `other_reference` del test:

```blade
<td class="det-ref">
    {{ $det->reference_value ?: ($det->test->other_reference ?? '') }}
</td>
```

Esto asegura que:
- Si `reference_value` tiene contenido (construido al crear), se muestra
- Si está vacío, se muestra `other_reference` del test como fallback
- Para determinaciones creadas ANTES de esta versión, el fallback funciona sin migrar datos

### 6.2 — PDF de admisiones (`resources/views/lab/admissions/pdf-mpdf.blade.php`)

Misma lógica:

```blade
<td class="det-ref">
    {{ $at->reference_value ?: ($at->test->other_reference ?? '') }}
</td>
```
