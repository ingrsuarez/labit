# v1.66.0 — Dashboard ejecutivo financiero + reubicación del panel de RRHH

**VERSIÓN:** v1.66.0
**SLUG:** dashboard-financiero-rrhh
**DEPENDENCIAS:** ninguna (toca módulos ya estables: Ventas v2.1.0, Compras v2.2.0, Recibos v1.39+, OP v1.39.1, multi-empresa v2.0.0)
**ESTIMACIÓN:** 1 día (con doc de Designer entregado previamente)
**PRIORIDAD:** media — visibilidad ejecutiva de operaciones del mes (gap detectado por PM)

---

## Descripción

Reemplazar el contenido del dashboard principal (`/dashboard`) por un panel ejecutivo
**financiero** con 4 KPIs del mes en curso (Ventas netas, Compras netas, Ingresos,
Egresos), cada uno con su gráfico de barras de evolución a 12 meses y variación
% vs mes anterior.

El contenido actual de RRHH (KPIs, gráficos de empleados, cumpleaños, solicitudes,
top puestos, accesos rápidos) se **muda íntegro** a una nueva ruta `/rrhh` con su
propio controller y vista. La mudanza es 1:1, sin cambios funcionales — solo cambia
el header del panel.

Se agrega una entrada nueva en el sidebar admin "Recursos Humanos" para acceder al
panel mudado.

        $egresos  = $this->service->egresos($companyId);

## PASO 6 — Crear `FinancialDashboardService`

**Objetivo:** centralizar la lógica de cálculo y testabilidad.

```bash
mkdir -p app/Services
```

En `app/Services/FinancialDashboardService.php`:

```php
<?php

namespace App\Services;

use App\Models\CollectionReceipt;
use App\Models\CreditNote;
use App\Models\PaymentOrder;
use App\Models\PurchaseCreditNote;
use App\Models\PurchaseInvoice;
use App\Models\SalesInvoice;
use Carbon\Carbon;
use Illuminate\Support\Facades\DB;

class FinancialDashboardService
{
    /**
     * @return array{
     *   label: string,
     *   monthly: array<int, array{label: string, value: float, is_current: bool}>,
     *   current_total: float,
     *   previous_total: float,
     *   variation_percent: float|null
     * }
     */
    public function ventas(?int $companyId): array
    {
        $start = now()->subMonths(11)->startOfMonth();
        $end = now()->endOfMonth();

        $invoices = SalesInvoice::query()
            ->when($companyId, fn ($q) => $q->where('company_id', $companyId))
            ->whereBetween('issue_date', [$start, $end])
            ->select(
                DB::raw('YEAR(issue_date) as y'),
                DB::raw('MONTH(issue_date) as m'),
                DB::raw('SUM(total) as total')
            )
            ->groupBy('y', 'm')
            ->get()
            ->keyBy(fn ($r) => sprintf('%04d-%02d', $r->y, $r->m));

        $credits = CreditNote::query()
            ->when($companyId, fn ($q) => $q->whereHas('salesInvoice', fn ($s) => $s->where('company_id', $companyId)))
            ->whereBetween('issue_date', [$start, $end])
            ->select(
                DB::raw('YEAR(issue_date) as y'),
                DB::raw('MONTH(issue_date) as m'),
                DB::raw('SUM(total) as total')
            )
            ->groupBy('y', 'm')
            ->get()
            ->keyBy(fn ($r) => sprintf('%04d-%02d', $r->y, $r->m));

        return $this->buildSeries(
            label: 'Ventas',
            invoices: $invoices,
            negatives: $credits
        );
    }

    public function compras(?int $companyId): array
    {
        $start = now()->subMonths(11)->startOfMonth();
        $end = now()->endOfMonth();

        $invoices = PurchaseInvoice::query()
            ->when($companyId, fn ($q) => $q->where('company_id', $companyId))
            ->whereBetween('issue_date', [$start, $end])
            ->select(
                DB::raw('YEAR(issue_date) as y'),
                DB::raw('MONTH(issue_date) as m'),
                DB::raw('SUM(total) as total')
            )
            ->groupBy('y', 'm')
            ->get()
            ->keyBy(fn ($r) => sprintf('%04d-%02d', $r->y, $r->m));

        $credits = PurchaseCreditNote::query()
            ->when($companyId, fn ($q) => $q->whereHas('purchaseInvoice', fn ($p) => $p->where('company_id', $companyId)))
            ->whereBetween('issue_date', [$start, $end])
            ->select(
                DB::raw('YEAR(issue_date) as y'),
                DB::raw('MONTH(issue_date) as m'),
                DB::raw('SUM(total) as total')
            )
            ->groupBy('y', 'm')
            ->get()
            ->keyBy(fn ($r) => sprintf('%04d-%02d', $r->y, $r->m));

        return $this->buildSeries(
            label: 'Compras',
            invoices: $invoices,
            negatives: $credits
        );
    }

    public function ingresos(?int $companyId): array
    {
        $start = now()->subMonths(11)->startOfMonth();
        $end = now()->endOfMonth();

        // ⚠ Verificar: ¿CollectionReceipt tiene company_id directo?
        // Si NO: filtrar vía customer (ver handoff sección 2)
        // Si SÍ: usar where('company_id', $companyId) directo
        $query = CollectionReceipt::query()
            ->whereBetween('date', [$start, $end]);

        if ($companyId) {
            if (\Schema::hasColumn('collection_receipts', 'company_id')) {
                $query->where('company_id', $companyId);
            } else {
                $query->whereHas('customer', fn ($c) => $c->where('company_id', $companyId));
            }
        }

        $rows = $query
            ->select(
                DB::raw('YEAR(date) as y'),
                DB::raw('MONTH(date) as m'),
                DB::raw('SUM(total) as total')
            )
            ->groupBy('y', 'm')
            ->get()
            ->keyBy(fn ($r) => sprintf('%04d-%02d', $r->y, $r->m));

        return $this->buildSeries(label: 'Ingresos', invoices: $rows, negatives: collect());
    }

    public function egresos(?int $companyId): array
    {
        $start = now()->subMonths(11)->startOfMonth();
        $end = now()->endOfMonth();

        $rows = PaymentOrder::query()
            ->when($companyId, fn ($q) => $q->where('company_id', $companyId))
            ->whereBetween('date', [$start, $end])
            ->select(
                DB::raw('YEAR(date) as y'),
                DB::raw('MONTH(date) as m'),
                DB::raw('SUM(total) as total')
            )
            ->groupBy('y', 'm')
            ->get()
            ->keyBy(fn ($r) => sprintf('%04d-%02d', $r->y, $r->m));

        return $this->buildSeries(label: 'Egresos', invoices: $rows, negatives: collect());
    }

    private function buildSeries(string $label, $invoices, $negatives): array
    {
        $monthly = [];
        $currentKey = now()->format('Y-m');
        $prevKey = now()->subMonth()->format('Y-m');

        $currentTotal = 0;
        $previousTotal = 0;

        for ($i = 11; $i >= 0; $i--) {
            $date = now()->subMonths($i);
            $key = $date->format('Y-m');

            $positive = (float) ($invoices->get($key)->total ?? 0);
            $negative = (float) ($negatives->get($key)->total ?? 0);
            $value = $positive - $negative;

            $monthly[] = [
                'label' => $date->locale('es')->isoFormat('MMM YY'),
                'value' => $value,
                'is_current' => $key === $currentKey,
            ];

            if ($key === $currentKey) {
                $currentTotal = $value;
            }
            if ($key === $prevKey) {
                $previousTotal = $value;
            }
        }

        $variation = $this->calculateVariation($currentTotal, $previousTotal);

        return [
            'label' => $label,
            'monthly' => $monthly,
            'current_total' => $currentTotal,
            'previous_total' => $previousTotal,
            'variation_percent' => $variation,
        ];
    }

    /**
     * Devuelve null si no hay base de comparación (mes anterior 0 con corriente > 0).
     * Devuelve 0 si ambos son 0 (sin cambios).
     */
    private function calculateVariation(float $current, float $previous): ?float
    {
        if ((float) $previous === 0.0) {
            if ((float) $current === 0.0) {
                return 0.0;
            }
            return null;
        }

        return round((($current - $previous) / $previous) * 100, 1);
    }
}
```
