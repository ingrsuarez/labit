---

## Descripción

Dos mejoras al sistema de protocolos de muestras:

1. **Firma digital de validadores**: Los usuarios que validan protocolos pueden subir
   una imagen de firma (jpg, png, gif) desde su perfil. La firma aparece en el PDF
   del protocolo al validar, debajo de la línea de firma y encima del nombre.

2. **Nombre automático de PDF**: Al descargar o visualizar un PDF de protocolo, el
   archivo se nombra como `{tipo_muestra}-{cliente}-{fecha}.{nro_protocolo}.pdf`
   en lugar del actual `Protocolo_{nro}.pdf`.

### 2.2 — Accessor para URL de firma

```php
public function getSignatureUrlAttribute(): ?string
{
    if (!$this->signature_path) {
        return null;
    }
    return \Storage::disk('public')->url($this->signature_path);
}

public function getSignatureAbsolutePathAttribute(): ?string
{
    if (!$this->signature_path) {
        return null;
    }
    return \Storage::disk('public')->path($this->signature_path);
}
```

El accessor `signatureAbsolutePath` es necesario para mPDF que requiere rutas absolutas
del filesystem para `<img>`.

## PASO 4 — Controlador UserSignatureController

Crear `app/Http/Controllers/UserSignatureController.php`:

```php
<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use Illuminate\Support\Facades\Storage;

class UserSignatureController extends Controller
{
    public function update(Request $request)
    {
        $request->validate([
            'signature' => 'required|image|mimes:png,jpg,jpeg,gif|max:2048',
        ]);

        $user = $request->user();

        // Borrar firma anterior si existe
        if ($user->signature_path && Storage::disk('public')->exists($user->signature_path)) {
            Storage::disk('public')->delete($user->signature_path);
        }

        $path = $request->file('signature')->store('signatures', 'public');
        $user->update(['signature_path' => $path]);

        return back()->with('success', 'Firma actualizada correctamente.');
    }

    public function destroy(Request $request)
    {
        $user = $request->user();

        if ($user->signature_path && Storage::disk('public')->exists($user->signature_path)) {
            Storage::disk('public')->delete($user->signature_path);
        }

        $user->update(['signature_path' => null]);

        return back()->with('success', 'Firma eliminada.');
    }
}
```

**Patrón seguido:** Idéntico al `PayrollController::updateSignature()` / `deleteSignature()`
(disco `public`, carpeta `signatures/`, borrar anterior antes de guardar nueva).

## PASO 7 — Nombre automático de PDF

En `app/Http/Controllers/SampleController.php`, modificar los métodos `downloadPdf()`
y `viewPdf()`.

### 7.1 — Método helper para generar el nombre

Agregar un método privado al controller:

```php
private function generatePdfFilename(Sample $sample): string
{
    $parts = [
        $sample->sample_type ?? 'Protocolo',
        $sample->customer?->name ?? 'SinCliente',
        $sample->sampling_date
            ? \Carbon\Carbon::parse($sample->sampling_date)->format('Y-m-d')
            : now()->format('Y-m-d'),
    ];

    // Sanitizar: quitar acentos, reemplazar espacios y caracteres especiales
    $sanitized = collect($parts)->map(function ($part) {
        $clean = iconv('UTF-8', 'ASCII//TRANSLIT//IGNORE', $part);
        $clean = preg_replace('/[^A-Za-z0-9_-]/', '_', $clean);
        $clean = preg_replace('/_+/', '_', $clean);
        return trim($clean, '_');
    })->implode('-');

    return $sanitized . '.' . $sample->protocol_number . '.pdf';
}
```

**Ejemplo de salida:** `Agua-TRANSENER_SA-2026-03-15.2026-00029.pdf`

### 7.2 — Modificar downloadPdf()

Cambiar la última línea de `downloadPdf()`:

**Antes:**
```php
return $pdf->download('Protocolo_'.$sample->protocol_number.'.pdf');
```

**Después:**
```php
return $pdf->download($this->generatePdfFilename($sample));
```

### 7.3 — Modificar viewPdf()

Cambiar la última línea de `viewPdf()`:

**Antes:**
```php
return $pdf->stream('Protocolo_'.$sample->protocol_number.'.pdf');
```

**Después:**
```php
return $pdf->stream($this->generatePdfFilename($sample));
```
