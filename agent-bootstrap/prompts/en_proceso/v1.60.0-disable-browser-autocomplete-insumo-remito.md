# v1.60.0 — Deshabilitar autocomplete del navegador en buscador de insumos del remito

**VERSIÓN:** v1.60.0
**SLUG:** disable-browser-autocomplete-insumo-remito
**DEPENDENCIAS:** ninguna
**ESTIMACIÓN:** 10min
**PRIORIDAD:** alta (bug visual activo en producción)

---

## Descripción

Al escribir en el campo "Buscar insumo..." de la tabla de ítems del remito,
el navegador muestra su propio dropdown de autocompletado (historial de valores
escritos antes) encima del buscador AJAX propio de la aplicación, tapando los
resultados.

**Root cause:** el `<input>` de búsqueda no tiene `autocomplete="off"`, por lo
que el navegador lo trata como un campo autocompletable.

**Fix:** agregar `autocomplete="off"` al input en `create.blade.php` y `edit.blade.php`.

---

## PASO 0 — Contexto

Leer sin ejecutar:

- `resources/views/delivery-notes/create.blade.php` → líneas 154–167
  (input `placeholder="Buscar insumo..."`)
- `resources/views/delivery-notes/edit.blade.php` → líneas 170–180
  (input `placeholder="Buscar insumo..."`)

---

## PASO 1 — Agregar autocomplete="off" en create.blade.php

Archivo: `resources/views/delivery-notes/create.blade.php`

Localizar el input `placeholder="Buscar insumo..."` (línea ~164).

**ANTES:**
```html
<input type="text"
       x-show="!item.supply_id"
       x-model="searchText"
       @input.debounce.300ms="doSearch()"
       @focus="if (searchText.length >= 2) showResults = true"
       @keydown.arrow-down.prevent="highlightNext()"
       @keydown.arrow-up.prevent="highlightPrev()"
       @keydown.enter.prevent="onSupplySearchEnter()"
       @keydown.escape="showResults = false"
       @keydown.tab="onTab($event)"
       placeholder="Buscar insumo..."
       :required="!item.supply_id"
       :id="'item-supply-' + index"
       class="flex-1 min-w-0 rounded border-gray-300 text-sm focus:border-zinc-500 focus:ring-zinc-500">
```

**DESPUÉS:** agregar `autocomplete="off"` después de `type="text"`:
```html
<input type="text"
       autocomplete="off"
       x-show="!item.supply_id"
       x-model="searchText"
       @input.debounce.300ms="doSearch()"
       @focus="if (searchText.length >= 2) showResults = true"
       @keydown.arrow-down.prevent="highlightNext()"
       @keydown.arrow-up.prevent="highlightPrev()"
       @keydown.enter.prevent="onSupplySearchEnter()"
       @keydown.escape="showResults = false"
       @keydown.tab="onTab($event)"
       placeholder="Buscar insumo..."
       :required="!item.supply_id"
       :id="'item-supply-' + index"
       class="flex-1 min-w-0 rounded border-gray-300 text-sm focus:border-zinc-500 focus:ring-zinc-500">
```
