## Descripción

Desde el listado de insumos, un botón "Unificar con..." abre un modal donde
el usuario elige el insumo destino (B). Al confirmar, todas las referencias
de A se reasignan a B en una transacción atómica, el stock por sede se suma,
y A queda inactivo con el prefijo `[UNIFICADO]` en el nombre.

La operación es irreversible — el modal muestra un preview de cuántos
registros se afectarán antes de confirmar.

**Tablas que referencian `supply_id` a reasignar:**
- `stock_movements`
- `purchase_invoice_items`
- `purchase_order_items`
- `delivery_note_items`
- `purchase_quotation_request_items`
- `purchase_credit_note_items`
- `supply_lab_branch_stock` (merge por sede: suma cantidades)
- `supplies.stock` (campo denormalizado: A.stock se suma a B.stock)

## PASO 2 — Método `merge` en SupplyController

```php
public function merge(Request $request, Supply $supply): \Illuminate\Http\RedirectResponse
{
    $this->authorize('supplies.edit');

    $request->validate([
        'target_id' => [
            'required',
            'integer',
            'exists:supplies,id',
            function (string $attribute, mixed $value, \Closure $fail) use ($supply): void {
                if ((int) $value === $supply->id) {
                    $fail('El insumo destino debe ser diferente al origen.');
                }
            },
        ],
    ]);

    $target = Supply::findOrFail((int) $request->target_id);

    if (! $target->is_active) {
        return back()->with('error', 'El insumo destino está inactivo.');
    }

    \DB::transaction(function () use ($supply, $target): void {
        // 1. Reasignar referencias simples
        \DB::table('stock_movements')->where('supply_id', $supply->id)->update(['supply_id' => $target->id]);
        \DB::table('purchase_invoice_items')->where('supply_id', $supply->id)->update(['supply_id' => $target->id]);
        \DB::table('purchase_order_items')->where('supply_id', $supply->id)->update(['supply_id' => $target->id]);
        \DB::table('delivery_note_items')->where('supply_id', $supply->id)->update(['supply_id' => $target->id]);
        \DB::table('purchase_quotation_request_items')->where('supply_id', $supply->id)->update(['supply_id' => $target->id]);
        \DB::table('purchase_credit_note_items')->where('supply_id', $supply->id)->whereNotNull('supply_id')->update(['supply_id' => $target->id]);

        // 2. Merge stock por sede
        $sourceBranchStocks = \DB::table('supply_lab_branch_stock')
            ->where('supply_id', $supply->id)
            ->get();

        foreach ($sourceBranchStocks as $row) {
            $existing = \DB::table('supply_lab_branch_stock')
                ->where('supply_id', $target->id)
                ->where('lab_branch_id', $row->lab_branch_id)
                ->first();

            if ($existing) {
                \DB::table('supply_lab_branch_stock')
                    ->where('supply_id', $target->id)
                    ->where('lab_branch_id', $row->lab_branch_id)
                    ->update(['quantity' => $existing->quantity + $row->quantity]);
                \DB::table('supply_lab_branch_stock')
                    ->where('supply_id', $supply->id)
                    ->where('lab_branch_id', $row->lab_branch_id)
                    ->delete();
            } else {
                \DB::table('supply_lab_branch_stock')
                    ->where('supply_id', $supply->id)
                    ->where('lab_branch_id', $row->lab_branch_id)
                    ->update(['supply_id' => $target->id]);
            }
        }

        // 3. Sumar stock global (campo denormalizado)
        $target->increment('stock', (float) $supply->stock);

        // 4. Actualizar last_price si A tiene precio más reciente
        if ((float) $supply->last_price > (float) $target->last_price) {
            $target->update(['last_price' => $supply->last_price]);
        }

        // 5. Desactivar A con prefijo identificador
        $supply->update([
            'is_active' => false,
            'name'      => '[UNIFICADO→' . $target->code . '] ' . $supply->name,
        ]);
    });

    return redirect()->route('supplies.index')
        ->with('success', "Insumo \"{$supply->name}\" unificado en \"{$target->name}\" correctamente.");
}
```

### 4.2 — Estado Alpine.js del modal

En el bloque `x-data` del `<div>` principal de `supplies/index.blade.php`, agregar:

```js
mergeModal: false,
mergeSource: { id: null, name: '', code: '' },
mergeTarget: null,
mergeSearch: '',
mergeSearchResults: [],
mergeSearchLoading: false,
mergePreviewData: null,
mergePreviewLoading: false,
mergeSubmitting: false,

openMergeModal(id, name, code) {
    this.mergeSource = { id, name, code };
    this.mergeTarget = null;
    this.mergeSearch = '';
    this.mergeSearchResults = [];
    this.mergePreviewData = null;
    this.mergeModal = true;
},

async searchMergeTarget() {
    if (this.mergeSearch.length < 2) { this.mergeSearchResults = []; return; }
    this.mergeSearchLoading = true;
    try {
        const resp = await fetch(`{{ route('supplies.search') }}?q=${encodeURIComponent(this.mergeSearch)}`, {
            headers: { 'Accept': 'application/json' }
        });
        const data = await resp.json();
        this.mergeSearchResults = data.filter(s => s.id !== this.mergeSource.id);
    } catch (e) {
        this.mergeSearchResults = [];
    } finally {
        this.mergeSearchLoading = false;
    }
},

async selectMergeTarget(supply) {
    this.mergeTarget = supply;
    this.mergeSearch = supply.name;
    this.mergeSearchResults = [];
    this.mergePreviewLoading = true;
    try {
        const resp = await fetch(`/supplies/${this.mergeSource.id}/merge-preview`, {
            headers: { 'Accept': 'application/json' }
        });
        this.mergePreviewData = await resp.json();
    } catch (e) {
        this.mergePreviewData = null;
    } finally {
        this.mergePreviewLoading = false;
    }
},
```

## PASO 5 — Test Feature

Crear `tests/Feature/SupplyMergeTest.php`:

```php
<?php

namespace Tests\Feature;

use App\Models\Supply;
use App\Models\User;
use Illuminate\Foundation\Testing\RefreshDatabase;
use Tests\TestCase;

class SupplyMergeTest extends TestCase
{
    use RefreshDatabase;

    public function test_merge_reasigna_stock_movements_y_desactiva_origen(): void
    {
        $user = User::factory()->withRole('admin')->create();
        $source = Supply::factory()->create(['stock' => 10, 'is_active' => true]);
        $target = Supply::factory()->create(['stock' => 5,  'is_active' => true]);

        \DB::table('stock_movements')->insert([
            'supply_id'    => $source->id,
            'type'         => 'entrada',
            'quantity'     => 10,
            'reason'       => 'compra',
            'user_id'      => $user->id,
            'occurred_at'  => now(),
            'created_at'   => now(),
            'updated_at'   => now(),
        ]);

        $response = $this->actingAs($user)->post(route('supplies.merge', $source), [
            'target_id' => $target->id,
        ]);

        $response->assertRedirect(route('supplies.index'));

        // Stock sumado en target
        $this->assertEquals(15, (float) $target->fresh()->stock);

        // Movimiento reasignado
        $this->assertDatabaseHas('stock_movements', [
            'supply_id' => $target->id,
        ]);
        $this->assertDatabaseMissing('stock_movements', [
            'supply_id' => $source->id,
        ]);

        // Origen desactivado
        $this->assertFalse($source->fresh()->is_active);
        $this->assertStringStartsWith('[UNIFICADO', $source->fresh()->name);
    }

    public function test_merge_rechaza_mismo_insumo(): void
    {
        $user = User::factory()->withRole('admin')->create();
        $supply = Supply::factory()->create();

        $response = $this->actingAs($user)->post(route('supplies.merge', $supply), [
            'target_id' => $supply->id,
        ]);

        $response->assertSessionHasErrors('target_id');
    }

    public function test_merge_rechaza_destino_inactivo(): void
    {
        $user = User::factory()->withRole('admin')->create();
        $source = Supply::factory()->create(['is_active' => true]);
        $target = Supply::factory()->create(['is_active' => false]);

        $response = $this->actingAs($user)->post(route('supplies.merge', $source), [
            'target_id' => $target->id,
        ]);

        $response->assertSessionHas('error');
    }
}
```
