# v1.56.0 — Botón "Crear Factura de Compra" desde el show y el index del remito

**VERSIÓN:** v1.56.0
**SLUG:** boton-crear-fc-desde-remito
**DEPENDENCIAS:** v1.36.0 (ruta purchase-invoices.create?delivery_note_id ya existe)
**ESTIMACIÓN:** 15min
**PRIORIDAD:** alta

---

## Descripción

Agregar un botón/link "Crear Factura de Compra" en la vista `delivery-notes/show` y una
acción rápida en `delivery-notes/index` que lleva directamente a crear una FC con los
ítems del remito precargados.

La ruta `purchase-invoices/create?delivery_note_id={id}` ya existe y funciona: el
`PurchaseInvoiceController::create()` detecta el parámetro, carga el remito con sus
ítems, y los precarga en el formulario. Solo falta el punto de entrada desde el remito.

El botón/link solo aparece cuando:
- El remito **no tiene** FC asociada aún (`!$deliveryNote->hasPurchaseInvoice()`)
- El usuario tiene permiso `purchase-invoices.create`

## PASO 1 — Botón en delivery-notes/show.blade.php

Archivo: `resources/views/delivery-notes/show.blade.php`

Localizar el bloque de acciones del header (el `<div class="flex items-center gap-3">` que
contiene el botón "Editar" y el link "← Volver").

Agregar el botón "Crear Factura de Compra" **antes** del botón "Editar", dentro del mismo
`@if(!$deliveryNote->hasPurchaseInvoice())`:

**ANTES:**
```blade
<div class="flex items-center gap-3">
    @if(!$deliveryNote->hasPurchaseInvoice())
        <a href="{{ route('delivery-notes.edit', $deliveryNote) }}"
           class="inline-flex items-center px-4 py-2 bg-zinc-700 text-white text-sm font-medium rounded-lg hover:bg-zinc-800 transition-colors">
            Editar
        </a>
    @endif
    <a href="{{ route('delivery-notes.index') }}" class="text-gray-500 hover:text-gray-700 text-sm font-medium">&larr; Volver</a>
</div>
```

**DESPUÉS:**
```blade
<div class="flex items-center gap-3">
    @if(!$deliveryNote->hasPurchaseInvoice())
        @can('purchase-invoices.create')
            <a href="{{ route('purchase-invoices.create', ['delivery_note_id' => $deliveryNote->id]) }}"
               class="inline-flex items-center gap-1.5 px-4 py-2 bg-teal-600 text-white text-sm font-medium rounded-lg hover:bg-teal-700 transition-colors">
                <svg class="w-4 h-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M9 12h6m-6 4h6m2 5H7a2 2 0 01-2-2V5a2 2 0 012-2h5.586a1 1 0 01.707.293l5.414 5.414a1 1 0 01.293.707V19a2 2 0 01-2 2z"/>
                </svg>
                Crear Factura de Compra
            </a>
        @endcan
        <a href="{{ route('delivery-notes.edit', $deliveryNote) }}"
           class="inline-flex items-center px-4 py-2 bg-zinc-700 text-white text-sm font-medium rounded-lg hover:bg-zinc-800 transition-colors">
            Editar
        </a>
    @endif
    <a href="{{ route('delivery-notes.index') }}" class="text-gray-500 hover:text-gray-700 text-sm font-medium">&larr; Volver</a>
</div>
```

**Verificación:** En el show de un remito sin FC asociada, aparece el botón verde "Crear Factura de Compra". Al hacer click, abre el formulario de nueva FC con el proveedor y los ítems del remito precargados. En un remito que ya tiene FC, el botón no aparece.

## PASO 2 — Acción en delivery-notes/index.blade.php

Archivo: `resources/views/delivery-notes/index.blade.php`

Localizar el bloque de acciones por fila (donde están "Ver", "Editar" y el form de eliminar).
Agregar el link "Crear FC" entre "Ver" y "Editar", condicionado a `!$hasInvoice`:

```blade
@if(!$hasInvoice)
    @can('purchase-invoices.create')
        <a href="{{ route('purchase-invoices.create', ['delivery_note_id' => $note->id]) }}"
           class="text-teal-600 hover:text-teal-800 text-xs font-medium">
            Crear FC
        </a>
    @endcan
@endif
```

Agregarlo en el bloque de acciones, manteniendo la separación visual consistente con
los demás links de la fila (misma clase `text-xs` que "Ver").

**Verificación:** En el listado de remitos, los que no tienen FC muestran el link "Crear FC" entre las acciones. Los que ya tienen FC no lo muestran.
