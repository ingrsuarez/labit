# v1.11.0 — Importación de obras sociales desde Excel

**VERSIÓN:** v1.11.0  
**SLUG:** importacion-obras-sociales-excel  
**DEPENDENCIAS:** v1.10.0 (nomencladores base deben existir en BD)  
**ESTIMACIÓN:** 30min  
**PRIORIDAD:** alta

---

## Descripción

Seeder que lee el archivo `docs/obras sociales.xlsx` (109 filas) y crea cada obra social
como `Insurance` con `type='obra_social'`, asignando el nomenclador base correspondiente
(`nomenclator_id`) y el valor PMO como `nbu_value`.

El seeder ya fue creado: `database/seeders/ObrasSocialesExcelSeeder.php`.

### Estructura del Excel

| Columna | Campo | Mapeo en Insurance |
|---|---|---|
| A | Sigla | Se guarda en `instructions` como referencia |
| B | Obra social (nombre completo + plan) | `name` |
| C | CUIT | `tax_id` |
| D | NOMENCLADOR | `nomenclator_id` (busca el Insurance base por nombre) |
| E | PMO | `nbu_value` (valor monetario de 1 NBU) |

### Mapeo nomencladores (columna D → Insurance base)

| Valor en Excel | Insurance name (type=nomenclador) |
|---|---|
| ISSN2020 | ISSN |
| 2016Uni | Nomenclador 2016 Uni |
| 20162023 | Nomenclador 20162023 |
| PAMI | PAMI |
| Omint | OMINT |
| Medicus | Medicus |
| 2012Reduc | Nomenclador 2012 Reducido |
| 2016 | Nomenclador 2016 |
| SWISS2012 | Swiss Medical |

## Instrucciones de producción

```bash
# 1. Primero los nomencladores base (si no se ejecutó v1.10.0)

# 2. Después las obras sociales
php artisan db:seed --class=ObrasSocialesExcelSeeder

php artisan view:clear
```

No requiere migraciones. Los archivos `.xlsx` deben estar en `docs/`.
