# v1.51.0 — Endpoint POST `/api/v1/results/batch` para ingesta de resultados desde LISCOM

**VERSIÓN:** v1.51.0
**SLUG:** api-ingesta-resultados-batch
**REPO:** `c:\wamp64\www\labit` (Laravel)
**DEPENDENCIAS:** v1.46.0 (auth API key) + v1.47.0 (endpoints de protocolos)
**ESTIMACIÓN:** 1 día
**PRIORIDAD:** alta — destraba v1.52.0 (cola de envío en LISCOM)

---

## Descripción

Crear el endpoint que recibe los resultados que LISCOM ya validó humanamente (v1.50.0) y los persiste contra los registros de resultado existentes en labit (`admission_tests`, `sample_determinations`, `vet_admission_tests`). El endpoint es **batch**, **idempotente**, y **respeta la validación del bioquímico**: si una determinación ya está validada por un usuario humano de labit, el valor entrante NO se sobrescribe — se rechaza el ítem y se devuelve el detalle para que LISCOM lo registre.

Esta versión NO tiene UI nueva (la pantalla de monitoreo viene en v1.53.0). Solo:

- Endpoint REST `POST /api/v1/results/batch` con auth API key (de v1.46.0).
- 2 modelos nuevos: `ResultBatch` (cabecera) + `ResultIngestion` (1 fila por OBX procesado).
- Servicio `ApiResultIngestionService` que orquesta lookup de protocolo + verificación `is_validated` + persistencia + audit.
- Códigos de status detallados por ítem para que LISCOM (v1.52.0) sepa qué reintentar y qué descartar.
- Tests Feature cubriendo todos los caminos.

## Decisiones registradas (sesión PM 2026-04-18)

| # | Pregunta | Decisión |
|---|---|---|
| 1 | Scope del endpoint | **Batch** (POST `/api/v1/results/batch`). Más eficiente para conexión inestable. No habrá endpoint individual |
| 2 | Estructura del payload | **JSON estructurado**. LISCOM ya parseó y matcheó. NO se manda HL7 raw a labit (audit forense queda en LISCOM) |
| 3 | Auto-validación | **Ingesta simple**: el resultado entra sin `is_validated`. El bioquímico valida en labit como hoy |
| 4 | Tipos de protocolo | **Unificado**: 1 endpoint detecta tipo por prefijo del `protocol_number` (`C-`/`S-`/`V-`) y ramifica internamente |
| 5 | Idempotencia | Por `(api_client_id, hl7_control_id)`. Mismo control_id = mismo resultado, devuelve el procesado original |
| 6 | UI admin | Ninguna en esta versión. La de monitoreo es v1.53.0 |
| 7 | **Si determinación YA validada por bioquímico** | **Rechazar ítem** con `ALREADY_VALIDATED`. Devolver `validated_at` + `validated_by`. NO sobrescribir |
| 8 | Si determinación tiene valor previo NO validado | **Sobrescribir + audit log** del cambio (overwrite explícito) |
| 9 | Si determinación no existe / test_id mismatch | Rechazar ítem con `DETERMINATION_NOT_FOUND` o `TEST_ID_MISMATCH` |

## PASO 3 — Servicio `ApiResultIngestionService`

Crear `app/Services/Api/ApiResultIngestionService.php`. Es la pieza central de la versión.

```php
<?php

namespace App\Services\Api;

use App\Models\Admission;
use App\Models\AdmissionTest;
use App\Models\ApiClient;
use App\Models\ResultBatch;
use App\Models\ResultIngestion;
use App\Models\Sample;
use App\Models\SampleDetermination;
use App\Models\VetAdmission;
use App\Models\VetAdmissionTest;
use Illuminate\Support\Facades\DB;
use Illuminate\Support\Facades\Log;

class ApiResultIngestionService
{
    public const STATUS_INGESTED   = 'ingested';
    public const STATUS_OVERWRITTEN = 'overwritten';
    public const STATUS_REJECTED   = 'rejected';
    public const STATUS_DUPLICATE  = 'duplicate';

    public const REASON_ALREADY_VALIDATED   = 'ALREADY_VALIDATED';
    public const REASON_DETERMINATION_NOT_FOUND = 'DETERMINATION_NOT_FOUND';
    public const REASON_TEST_ID_MISMATCH    = 'TEST_ID_MISMATCH';
    public const REASON_PROTOCOL_NOT_FOUND  = 'PROTOCOL_NOT_FOUND';
    public const REASON_PROTOCOL_OUT_OF_BRANCH = 'PROTOCOL_OUT_OF_BRANCH';

    /**
     * Procesa un batch completo. Devuelve estructura para serializar al cliente.
     */
    public function process(ApiClient $client, array $payload): array
    {
        return DB::transaction(function () use ($client, $payload) {
            // Idempotencia a nivel de batch: mismo external_batch_id = devolver el original
            $existingBatch = ResultBatch::where('api_client_id', $client->id)
                ->where('external_batch_id', $payload['batch_id'])
                ->first();

            if ($existingBatch) {
                Log::info('api.results.batch.duplicate', [
                    'api_client' => $client->id,
                    'batch_id' => $payload['batch_id'],
                ]);
                return $this->serializeExistingBatch($existingBatch);
            }

            $batch = ResultBatch::create([
                'api_client_id' => $client->id,
                'external_batch_id' => $payload['batch_id'],
                'source_app' => 'LISCOM',
                'items_total' => count($payload['items']),
                'raw_request' => $this->truncatePayload($payload),
            ]);

            $itemsResponse = [];
            $counters = ['ingested' => 0, 'overwritten' => 0, 'rejected' => 0, 'duplicate' => 0];

            foreach ($payload['items'] as $item) {
                $itemResult = $this->processItem($client, $batch, $item);
                $itemsResponse[] = $itemResult['response'];
                $counters[$itemResult['counter']]++;
            }

            $batch->update([
                'items_ingested' => $counters['ingested'],
                'items_overwritten' => $counters['overwritten'],
                'items_rejected' => $counters['rejected'],
                'items_duplicate' => $counters['duplicate'],
            ]);

            Log::info('api.results.batch.processed', [
                'api_client' => $client->id,
                'batch_id' => $payload['batch_id'],
                'counters' => $counters,
            ]);

            return [
                'batch_id' => $payload['batch_id'],
                'items' => $itemsResponse,
            ];
        });
    }

    private function processItem(ApiClient $client, ResultBatch $batch, array $item): array
    {
        // Idempotencia por mensaje: (api_client, hl7_control_id)
        $existingIngestion = ResultIngestion::where('api_client_id', $client->id)
            ->where('hl7_control_id', $item['hl7_control_id'])
            ->first();

        if ($existingIngestion) {
            return [
                'counter' => 'duplicate',
                'response' => [
                    'external_message_id' => $item['external_message_id'] ?? null,
                    'hl7_control_id' => $item['hl7_control_id'],
                    'status' => self::STATUS_DUPLICATE,
                    'original_batch_id' => $existingIngestion->batch->external_batch_id,
                    'results' => $existingIngestion->items_summary,
                ],
            ];
        }

        // Resolver protocolo y verificar pertenencia a la sede del API client
        $protocolNumber = $item['protocol_number'];
        $protocol = $this->resolveProtocol($protocolNumber);

        if (!$protocol) {
            return $this->rejectMessage(
                $client, $batch, $item, self::REASON_PROTOCOL_NOT_FOUND
            );
        }

        // Verificar que el protocolo pertenezca a la sede del API client
        if (isset($protocol->lab_branch_id) && $protocol->lab_branch_id !== $client->lab_branch_id) {
            return $this->rejectMessage(
                $client, $batch, $item, self::REASON_PROTOCOL_OUT_OF_BRANCH
            );
        }

        // Procesar cada result individualmente
        $resultsSummary = [];
        $allRejected = true;

        foreach ($item['results'] as $result) {
            $itemSummary = $this->processSingleResult($protocol['type'], $protocol['model'], $result);
            $resultsSummary[] = $itemSummary;

            if (in_array($itemSummary['status'], [self::STATUS_INGESTED, self::STATUS_OVERWRITTEN], true)) {
                $allRejected = false;
            }
        }

        $messageStatus = $allRejected ? self::STATUS_REJECTED : self::STATUS_INGESTED;
        if (!$allRejected && collect($resultsSummary)->contains('status', self::STATUS_REJECTED)) {
            $messageStatus = 'partial';
        }

        ResultIngestion::create([
            'result_batch_id' => $batch->id,
            'api_client_id' => $client->id,
            'external_message_id' => $item['external_message_id'] ?? null,
            'hl7_control_id' => $item['hl7_control_id'],
            'protocol_number' => $protocolNumber,
            'protocol_type' => $protocol['type'],
            'equipment_name' => $item['equipment_name'] ?? null,
            'status' => $messageStatus === 'partial' ? self::STATUS_INGESTED : $messageStatus,
            'items_summary' => $resultsSummary,
        ]);

        return [
            'counter' => $allRejected ? 'rejected' : 'ingested',
            'response' => [
                'external_message_id' => $item['external_message_id'] ?? null,
                'hl7_control_id' => $item['hl7_control_id'],
                'status' => $messageStatus,
                'protocol_number' => $protocolNumber,
                'results' => $resultsSummary,
            ],
        ];
    }

    /**
     * Detecta tipo por prefijo del protocol_number (consistente con v1.47.0).
     * Devuelve ['type' => 'clinical'|'sample'|'vet', 'model' => instancia del protocolo] o null.
     */
    private function resolveProtocol(string $protocolNumber): ?array
    {
        $prefix = strtoupper(substr($protocolNumber, 0, 2));

        return match ($prefix) {
            'C-' => $this->loadProtocol(Admission::class, $protocolNumber, 'clinical', 'admissionTests'),
            'S-' => $this->loadProtocol(Sample::class, $protocolNumber, 'sample', 'sampleDeterminations'),
            'V-' => $this->loadProtocol(VetAdmission::class, $protocolNumber, 'vet', 'vetAdmissionTests'),
            default => null,
        };
    }

    private function loadProtocol(string $modelClass, string $protocolNumber, string $type, string $relation): ?array
    {
        $model = $modelClass::where('protocol_number', $protocolNumber)->first();
        return $model ? ['type' => $type, 'model' => $model, 'relation' => $relation] : null;
    }

    /**
     * Procesa un result individual contra el protocolo. Acá vive la regla principal:
     * NO sobrescribir si is_validated = true.
     */
    private function processSingleResult(string $protocolType, $protocol, array $result): array
    {
        // Buscar la determinación específica
        $determination = $this->findDetermination($protocolType, $protocol, $result['labit_test_id']);

        if (!$determination) {
            return [
                'obx_index' => $result['obx_index'],
                'status' => self::STATUS_REJECTED,
                'reason' => self::REASON_DETERMINATION_NOT_FOUND,
                'labit_test_id' => $result['labit_test_id'],
            ];
        }

        // ⚠️ REGLA CRÍTICA: si ya está validado por bioquímico, no tocar
        if ($determination->is_validated) {
            Log::info('api.results.item.rejected_already_validated', [
                'protocol_type' => $protocolType,
                'determination_id' => $determination->id,
                'incoming_value' => $result['value'],
                'existing_value' => $determination->result ?? $determination->value,
                'validated_by' => $determination->validated_by,
                'validated_at' => $determination->validated_at?->toIso8601String(),
            ]);

            return [
                'obx_index' => $result['obx_index'],
                'status' => self::STATUS_REJECTED,
                'reason' => self::REASON_ALREADY_VALIDATED,
                'determination_id' => $determination->id,
                'validated_at' => $determination->validated_at?->toIso8601String(),
                'validated_by_name' => $determination->validatedBy?->name,
            ];
        }

        // Detectar overwrite (había valor previo no validado)
        $hadValue = !empty($determination->result ?? $determination->value);
        $previousValue = $hadValue ? ($determination->result ?? $determination->value) : null;

        // Persistir
        $this->persistResult($determination, $result);

        $finalStatus = $hadValue ? self::STATUS_OVERWRITTEN : self::STATUS_INGESTED;

        if ($hadValue) {
            Log::info('api.results.item.overwritten', [
                'protocol_type' => $protocolType,
                'determination_id' => $determination->id,
                'previous_value' => $previousValue,
                'new_value' => $result['value'],
            ]);
        }

        return [
            'obx_index' => $result['obx_index'],
            'status' => $finalStatus,
            'determination_id' => $determination->id,
            'previous_value' => $previousValue,
        ];
    }

    private function findDetermination(string $protocolType, $protocol, int $testId)
    {
        return match ($protocolType) {
            'clinical' => AdmissionTest::where('admission_id', $protocol->id)->where('test_id', $testId)->first(),
            'sample' => SampleDetermination::where('sample_id', $protocol->id)->where('test_id', $testId)->first(),
            'vet' => VetAdmissionTest::where('vet_admission_id', $protocol->id)->where('test_id', $testId)->first(),
        };
    }

    /**
     * Persiste el valor en el campo correcto según el tipo de determinación.
     * El Dev debe verificar el nombre real del campo de valor en cada modelo.
     */
    private function persistResult($determination, array $result): void
    {
        // El campo del valor varía por modelo: AdmissionTest tiene `result`,
        // SampleDetermination tiene `result`, VetAdmissionTest tiene `result`.
        // Verificar al implementar; ajustar si difiere.
        $determination->result = $result['value'];

        // Campos opcionales si existen en el modelo (skip si no)
        if (in_array('unit', $determination->getFillable(), true) && isset($result['unit'])) {
            $determination->unit = $result['unit'];
        }
        if (in_array('analyzed_at', $determination->getFillable(), true)) {
            $determination->analyzed_at = now();
        }
        // observations: append nota indicando origen API
        if (in_array('observations', $determination->getFillable(), true)) {
            $current = $determination->observations ?? '';
            $note = "[API LISCOM " . now()->format('Y-m-d H:i') . "]";
            $determination->observations = trim($current . "\n" . $note);
        }

        $determination->save();
    }

    private function rejectMessage(ApiClient $client, ResultBatch $batch, array $item, string $reason): array
    {
        ResultIngestion::create([
            'result_batch_id' => $batch->id,
            'api_client_id' => $client->id,
            'external_message_id' => $item['external_message_id'] ?? null,
            'hl7_control_id' => $item['hl7_control_id'],
            'protocol_number' => $item['protocol_number'],
            'equipment_name' => $item['equipment_name'] ?? null,
            'status' => self::STATUS_REJECTED,
            'rejection_reason' => $reason,
            'items_summary' => [],
        ]);

        return [
            'counter' => 'rejected',
            'response' => [
                'external_message_id' => $item['external_message_id'] ?? null,
                'hl7_control_id' => $item['hl7_control_id'],
                'status' => self::STATUS_REJECTED,
                'reason' => $reason,
                'protocol_number' => $item['protocol_number'],
            ],
        ];
    }

    private function serializeExistingBatch(ResultBatch $batch): array
    {
        return [
            'batch_id' => $batch->external_batch_id,
            'duplicate' => true,
            'items' => $batch->ingestions->map(fn ($ing) => [
                'external_message_id' => $ing->external_message_id,
                'hl7_control_id' => $ing->hl7_control_id,
                'status' => $ing->status,
                'protocol_number' => $ing->protocol_number,
                'results' => $ing->items_summary,
            ])->all(),
        ];
    }

    private function truncatePayload(array $payload): array
    {
        // Limitar a ~64KB de raw request para no saturar la BD
        $encoded = json_encode($payload);
        if (strlen($encoded) <= 64 * 1024) {
            return $payload;
        }
        return ['_truncated' => true, 'batch_id' => $payload['batch_id'] ?? null, 'items_count' => count($payload['items'] ?? [])];
    }
}
```

> **Verificación importante para el Dev:** los campos exactos de cada modelo de determinación (`result` vs `value`, `analyzed_at`, `observations`, etc.) pueden diferir. Revisar cada `$fillable` y ajustar `persistResult` si es necesario. Lo central que NO debe cambiar es el check `if ($determination->is_validated) { reject }`.

namespace App\Http\Controllers\Api\V1;
