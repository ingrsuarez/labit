## Descripción

Agrega la capacidad de escanear o subir una imagen del QR de una factura recibida (formato RG 4892 de AFIP)
para autocompletar el formulario de factura de compra. También agrega campos AFIP al modelo de compras
para registrar el CAE del proveedor y mantener trazabilidad fiscal.

El QR de las facturas electrónicas argentinas codifica una URL `https://www.afip.gob.ar/fe/qr/?p=BASE64_JSON`
donde el JSON contiene: cuit emisor, tipo de comprobante, número, punto de venta, importe, fecha, CAE, etc.

### 3.2 — Lógica Alpine.js del escáner

```js
function qrScanner() {
    return {
        cameraActive: false,
        scanner: null,
        scanResult: null,
        scanError: null,

        // Decodificar QR AFIP: URL → base64 → JSON
        decodeAfipQr(decodedText) {
            this.scanError = null;
            try {
                // El QR contiene una URL: https://www.afip.gob.ar/fe/qr/?p=BASE64
                let base64Data;
                if (decodedText.includes('afip.gob.ar/fe/qr')) {
                    const url = new URL(decodedText);
                    base64Data = url.searchParams.get('p');
                } else {
                    // Intentar como base64 directo
                    base64Data = decodedText;
                }

                if (!base64Data) throw new Error('QR no contiene datos AFIP válidos');

                const json = JSON.parse(atob(base64Data));

                // Mapear tipoCmp a letra
                const tipoMap = {1: 'A', 2: 'A', 3: 'A', 6: 'B', 7: 'B', 8: 'B', 11: 'C', 12: 'C', 13: 'C'};
                // 1=Factura A, 6=Factura B, 11=Factura C, 3=NC A, 8=NC B, 13=NC C
                const tipoCmpLabels = {
                    1: 'Factura A', 2: 'ND A', 3: 'NC A',
                    6: 'Factura B', 7: 'ND B', 8: 'NC B',
                    11: 'Factura C', 12: 'ND C', 13: 'NC C'
                };

                this.scanResult = {
                    ...json,
                    tipoCmpLabel: tipoCmpLabels[json.tipoCmp] || `Tipo ${json.tipoCmp}`,
                    voucherType: tipoMap[json.tipoCmp] || 'A',
                    ptoVta: String(json.ptoVta).padStart(5, '0'),
                    nroCmp: String(json.nroCmp).padStart(8, '0'),
                };
            } catch (e) {
                this.scanError = 'No se pudo leer el QR. Asegurate de que sea el QR de una factura electrónica AFIP.';
                this.scanResult = null;
            }
        },

        startCamera() {
            this.cameraActive = true;
            this.$nextTick(() => {
                this.scanner = new Html5QrcodeScanner('qr-reader', {
                    fps: 10,
                    qrbox: { width: 250, height: 250 },
                    rememberLastUsedCamera: true,
                });
                this.scanner.render(
                    (decodedText) => {
                        this.decodeAfipQr(decodedText);
                        this.stopCamera();
                    },
                    (error) => { /* ignorar errores de frames sin QR */ }
                );
            });
        },

        stopCamera() {
            if (this.scanner) {
                this.scanner.clear().catch(() => {});
                this.scanner = null;
            }
            this.cameraActive = false;
        },

        async scanFromFile(event) {
            const file = event.target.files[0];
            if (!file) return;
            try {
                const html5Qr = new Html5Qrcode('qr-reader-file');
                const result = await html5Qr.scanFile(file, true);
                this.decodeAfipQr(result);
                html5Qr.clear();
            } catch (e) {
                this.scanError = 'No se encontró un QR válido en la imagen.';
            }
            event.target.value = '';
        },

        applyToForm() {
            if (!this.scanResult) return;
            const r = this.scanResult;

            // Setear campos del formulario
            // voucher_type (A/B/C)
            const voucherTypeEl = document.querySelector('[name="voucher_type"]');
            if (voucherTypeEl) { voucherTypeEl.value = r.voucherType; voucherTypeEl.dispatchEvent(new Event('change')); }

            // point_of_sale
            const posEl = document.querySelector('[name="point_of_sale"]');
            if (posEl) { posEl.value = r.ptoVta; }

            // invoice_number
            const numEl = document.querySelector('[name="invoice_number"]');
            if (numEl) { numEl.value = r.nroCmp; }

            // issue_date
            const dateEl = document.querySelector('[name="issue_date"]');
            if (dateEl && r.fecha) { dateEl.value = r.fecha; }

            // CAE (hidden)
            const caeEl = document.querySelector('[name="cae"]');
            if (caeEl) { caeEl.value = r.codAut; }

            // CUIT emisor (hidden)
            const cuitEl = document.querySelector('[name="cuit_emisor"]');
            if (cuitEl) { cuitEl.value = r.cuit; }

            // QR data completo (hidden)
            const qrDataEl = document.querySelector('[name="qr_data"]');
            if (qrDataEl) { qrDataEl.value = JSON.stringify(this.scanResult); }

            // Buscar proveedor por CUIT
            this.lookupSupplierByCuit(r.cuit);
        },

        async lookupSupplierByCuit(cuit) {
            try {
                const formatted = String(cuit);
                const response = await fetch(`/suppliers/by-cuit/${formatted}`);
                if (response.ok) {
                    const supplier = await response.json();
                    const supplierEl = document.querySelector('[name="supplier_id"]');
                    if (supplierEl && supplier.id) {
                        supplierEl.value = supplier.id;
                        supplierEl.dispatchEvent(new Event('change'));
                    }
                }
            } catch (e) {
                // No encontró proveedor — el usuario lo selecciona manualmente
            }
        }
    };
}
```

**Verificación:** Escanear QR con cámara → datos se muestran. Subir imagen con QR → datos se muestran. Click "Aplicar" → formulario se autocompleta.

```php
public function findByCuit(string $cuit)
{
    // Normalizar: sacar guiones
    $cuitNormalized = str_replace('-', '', $cuit);

    $supplier = Supplier::where('cuit', $cuit)
        ->orWhere('cuit', $cuitNormalized)
        ->orWhere('cuit', 'like', "%-{$cuitNormalized}%")
        ->first();

    if (!$supplier) {
        return response()->json(['message' => 'Proveedor no encontrado'], 404);
    }

    return response()->json([
        'id'   => $supplier->id,
        'name' => $supplier->name,
        'cuit' => $supplier->cuit,
    ]);
}
```

Verificar primero que el modelo `Supplier` tenga un campo `cuit`. Si no lo tiene, revisar
qué campo equivalente usa (puede ser `tax_id`, `document_number`, etc.) y adaptar la query.

## Notas / Consideraciones

- El QR de AFIP (RG 4892) codifica datos del comprobante pero NO los ítems. El desglose de ítems se carga manualmente.
- `html5-qrcode` funciona en HTTPS o localhost. En HTTP no-localhost la cámara no se activa (restricción del navegador). El upload de imagen funciona siempre.
- El mapeo de `tipoCmp` cubre los tipos más comunes (Factura/ND/NC A/B/C). Tipos menos frecuentes (Factura E, etc.) se muestran como "Tipo N".
- La búsqueda de proveedor por CUIT intenta varias normalizaciones (con/sin guiones) para maximizar matches.
- A futuro se podría validar el CAE contra AFIP (FECompConsultar) para verificar autenticidad.
