## Descripción

Eliminar la columna "Servicio" (select separado) de la tabla de ítems de factura de compra
y unificar la búsqueda en un único campo "Descripción / Insumo / Servicio".

El dropdown del buscador muestra dos secciones:
- **INSUMOS**: resultados server-side del endpoint `supplies.search`
- **SERVICIOS**: filtrado client-side de `purchaseServiceGroups`

Al seleccionar cualquiera de los dos, el input se oculta y aparece un badge diferenciado:
- Insumo → badge teal (igual que hoy)
- Servicio → badge indigo (nuevo)

Si se selecciona un servicio, los campos lote/vencimiento NO aparecen.

**Documento de diseño:** `agent-bootstrap/prompts/pendientes/DISEÑO_v1.55.0-buscador-unificado-fc.md`
**Mockups:** `assets/design-v1.55.0-dropdown-unificado.png`, `assets/design-v1.55.0-badges-estados.png`

### 1.2 — Reemplazar la función `supplySearch()` en el `<script>`

Eliminar la función `supplySearch()` completa y reemplazarla por `itemSearch()`.
La nueva función recibe `serviceGroups` como tercer parámetro.

Cambios respecto a la función actual:

**a) Nueva propiedad `serviceResults`** para los servicios filtrados client-side:
```js
serviceResults: [],
```

**b) Modificar `doSearch()`** para buscar en paralelo insumos (API) y servicios (client-side):
```js
async doSearch() {
    item.description = this.searchText;
    const q = this.searchText.trim().toLowerCase();
    if (q.length < 2) {
        this.results = [];
        this.serviceResults = [];
        this.showResults = false;
        return;
    }
    // Filtrado client-side de servicios
    this.serviceResults = [];
    for (const g of serviceGroups) {
        for (const s of g.services) {
            const haystack = ((s.code || '') + ' ' + s.name + ' ' + g.name).toLowerCase();
            if (haystack.includes(q)) {
                this.serviceResults.push({ ...s, _group_name: g.name });
            }
        }
        if (this.serviceResults.length >= 6) break;
    }
    // Búsqueda server-side de insumos (igual que antes)
    this.loading = true;
    try {
        const resp = await fetch(`{{ route('supplies.search') }}?q=${encodeURIComponent(this.searchText)}`);
        this.results = await resp.json();
        this.showResults = true;
        this.highlightedIndex = -1;
    } catch (e) {
        this.results = [];
    } finally {
        this.loading = false;
    }
},
```

**c) Agregar `selectService(svc)`:**
```js
selectService(svc) {
    item.purchase_service_id = String(svc.id);
    item.supply_id = '';
    item._supply_code = '';
    item._supply_label = '';
    item._service_label = (svc.code ? svc.code + ' — ' : '') + svc.name;
    item.description = item._service_label;
    item.lot_number = '';
    item.expiration_date = '';
    item.updates_stock = false;
    this.searchText = '';
    this.showResults = false;
    this.$nextTick(() => {
        const qtyInput = document.querySelector(`#item-qty-${index}`);
        if (qtyInput) qtyInput.focus();
    });
},
```

**d) Modificar `unlinkItem()` (renombrar `unlinkSupply()`):**
```js
unlinkItem() {
    item.supply_id = '';
    item.purchase_service_id = '';
    item._supply_code = '';
    item._supply_label = '';
    item._service_label = '';
    item.description = '';
    item.lot_number = '';
    item.expiration_date = '';
    this.searchText = '';
    this.$nextTick(() => {
        const el = document.querySelector(`#item-desc-${index}`);
        if (el) el.focus();
    });
},
```

**e) Actualizar `init()`** para manejar también servicios precargados:
```js
init() {
    if (item.supply_id && (item._supply_label || item.description)) {
        this.searchText = '';
    } else if (item.purchase_service_id && item.description) {
        item._service_label = item.description;
        this.searchText = '';
    }
}
```
