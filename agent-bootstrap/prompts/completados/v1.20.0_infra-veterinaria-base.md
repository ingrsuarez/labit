## Descripción

Crea la infraestructura base del módulo veterinario: tabla de especies con CRUD,
seeder de las 57 determinaciones veterinarias del Excel `docs/nomenclador veterinario.xlsx`,
y el nomenclador base "Veterinario" con los valores NBU correspondientes.

Las determinaciones se crean como registros en `tests` con `categories: ["veterinario"]`.
El nomenclador se crea como un `Insurance` con `type = 'nomenclador'` y sus prácticas
en `insurance_tests`, siguiendo el patrón existente de `NomencladoresExcelSeeder`.

## PASO 1 — Migración y modelo Species

Crear migración para la tabla `species`:

```bash
php artisan make:migration create_species_table
```

Schema:
```php
Schema::create('species', function (Blueprint $table) {
    $table->id();
    $table->string('name');          // Felino, Canino, Equino
    $table->string('code', 20);      // felino, canino, equino
    $table->boolean('is_active')->default(true);
    $table->timestamps();

    $table->unique('code');
});
```

Crear modelo `app/Models/Species.php`:
```php
class Species extends Model
{
    use HasFactory;

    protected $fillable = ['name', 'code', 'is_active'];

    protected $casts = [
        'is_active' => 'boolean',
    ];
}
```

**Verificación:** `php artisan migrate` sin errores.

## PASO 2 — Seeder de especies base

Crear `database/seeders/SpeciesSeeder.php`:

```php
class SpeciesSeeder extends Seeder
{
    public function run(): void
    {
        $species = [
            ['name' => 'Canino', 'code' => 'canino'],
            ['name' => 'Felino', 'code' => 'felino'],
            ['name' => 'Equino', 'code' => 'equino'],
        ];

        foreach ($species as $sp) {
            Species::firstOrCreate(['code' => $sp['code']], $sp);
        }
    }
}
```

Ejecutar: `php artisan db:seed --class=SpeciesSeeder`

**Verificación:** `php artisan tinker` → `Species::count()` retorna 3.

## PASO 3 — Seeder de determinaciones veterinarias y nomenclador

Crear `database/seeders/NomencladorVeterinarioSeeder.php`.

Seguir el patrón exacto de `NomencladoresExcelSeeder`:
1. Leer `docs/nomenclador veterinario.xlsx` con PhpSpreadsheet
2. Crear el `Insurance` nomenclador base con `name = 'Veterinario'`, `type = 'nomenclador'`, `nbu_value = 1`
3. Para cada fila del Excel (fila 2 en adelante):
   - Columna A: `Codigo` (código del test)
   - Columna B: `nbu` (unidades NBU)
   - Columna C: `Análisis` (nombre del test)
   - Columna D: `Muestra` (material/tipo de muestra)
4. Buscar el test por código. Si no existe, crearlo con `categories: ["veterinario"]`
5. Si el test ya existe (código compartido con clínico/aguas), agregar `"veterinario"` al array `categories` sin pisar las categorías existentes
6. Crear o actualizar la práctica en `insurance_tests`

**Datos del Excel** (referencia para el agente, las columnas son A=Codigo, B=nbu, C=Análisis, D=Muestra):
```
Fila 1 (header): Codigo | nbu | Análisis | Muestra
Fila 2: 200475 | 3 | Hemograma | Sangre c/edta
Fila 3: 200481 | 7 | Hepatograma | Suero
Fila 4: (vacío) | 4 | Hemograma + Plaquetas | Sangre c/edta
... (57 filas de datos, ver Excel para el listado completo)
```

**Importante:** La fila 4 no tiene código. Cuando el código está vacío, generar uno automático con prefijo `VET-` + número secuencial (ej: `VET-001`). Loguear con `$this->command->warn()` las filas sin código.

Ejecutar: `php artisan db:seed --class=NomencladorVeterinarioSeeder`

## Notas / Consideraciones

- Los nomencladores se modelan como `Insurance` con `type = 'nomenclador'`. No crear un modelo nuevo para esto.
- Las prácticas van en `insurance_tests` con `nbu_units` del Excel. El `nbu_value` del nomenclador arranca en 1 (se ajusta después desde la UI).
- Algunos tests del Excel pueden ya existir en la BD (mismos códigos que clínico o aguas). En ese caso, NO crear uno nuevo — solo agregar `"veterinario"` al array `categories` y crear la práctica en `insurance_tests`.
- La columna `material` del test no es un string sino una FK a la tabla `materials`. Si el material del Excel no matchea, dejarlo como null y loguear un warning.
- El Excel tiene una fila sin código (fila 4: "Hemograma + Plaquetas"). Generar código automático `VET-001`.
