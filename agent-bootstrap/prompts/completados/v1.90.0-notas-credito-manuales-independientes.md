# v1.90.0 — Notas de crédito manuales independientes

**Tipo:** minor — carga manual de NC sin factura asociada en el sistema
**Prioridad:** alta — bloquea la operatoria de registrar NC del sistema anterior
**Dependencias:** ninguna (v1.4.0 notas de crédito electrónicas ya completada)
**Estimación:** 2–3h

---

## Qué hace

Agrega la posibilidad de crear notas de crédito manualmente desde el listado de NC, sin
requerir una factura de venta existente en Labit. Pensado para cargar NC ya emitidas en otro
sistema con un punto de venta manual (no electrónico). No se conecta a AFIP.

El flujo actual (crear NC desde una factura electrónica) sigue funcionando igual.

## PASO 4 — CreditNoteController: métodos createManual y storeManual

En `app/Http/Controllers/CreditNoteController.php`:

### `createManual()`

```php
public function createManual()
{
    $customers = \App\Models\Customer::where('company_id', active_company_id())
        ->orderBy('name')
        ->get(['id', 'name', 'tax', 'cuit']);

    $pointsOfSale = \App\Models\PointOfSale::where('company_id', active_company_id())
        ->where('is_electronic', false)
        ->orderBy('code')
        ->get(['id', 'code', 'name']);

    return view('credit-notes.create-manual', compact('customers', 'pointsOfSale'));
}
```

### `storeManual()`

Validaciones:
- `customer_id` — required, exists:customers,id
- `voucher_type` — required, in:A,B,C
- `point_of_sale_id` — required, exists:points_of_sale,id (validar que sea no-electrónico)
- `credit_note_number` — required, string, max:20
- `issue_date` — required, date
- `reason` — required, string, max:500
- `items` — required, array, min:1
- `items.*.description` — required, string, max:255
- `items.*.quantity` — required, numeric, min:0.01
- `items.*.unit_price` — required, numeric, min:0
- `items.*.iva_rate` — required, numeric, in:0,10.5,21,27

Lógica del store:
1. Verificar que el PdV seleccionado NO sea electrónico (`abort_if($pos->is_electronic, 422)`)
2. Crear la NC con:
   - `company_id` = `active_company_id()`
   - `sales_invoice_id` = null
   - `is_electronic` = false
   - `status` = 'confirmada' (ya fue emitida en otro sistema)
   - `credit_note_number` = el número ingresado
   - `created_by` = Auth::id()
3. Crear los ítems con cálculo de IVA (mismo patrón que `store`)
4. Llamar `$creditNote->recalculate()`
5. Generar asiento contable via `recordCreditNoteJournalEntry()`
6. Redirect a `credit-notes.show` con mensaje de éxito

**Verificación:** Se puede crear una NC manual sin factura, queda con status "confirmada".
