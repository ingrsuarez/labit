# v1.32.3 — Marca en búsqueda y display de insumos

**VERSIÓN:** v1.32.3  
**SLUG:** marca-busqueda-display-insumos  
**DEPENDENCIAS:** v1.32.0 (buscador facturas de compra), v1.32.2 (buscador remitos)  
**ESTIMACIÓN:** 30min  
**PRIORIDAD:** media

---

## Descripción

Mejora el buscador de insumos en facturas de compra y remitos para incluir la marca
en tres lugares:

1. **Búsqueda**: el endpoint `supplies/search` ahora busca también por `brand`
2. **Dropdown de resultados**: muestra `Código — Nombre — Marca` en cada opción
3. **Display del ítem seleccionado**: el input y el badge muestran `Nombre — Marca`
   como texto unificado, sin elementos separados

El campo `brand` ya existe en la tabla `supplies`.

## PASO 1 — Actualizar el endpoint de búsqueda

En `SupplyController::search()`, agregar `brand` al criterio de búsqueda y a los campos retornados:

```php
public function search(Request $request)
{
    $q = $request->get('q', '');
    if (strlen($q) < 2) {
        return response()->json([]);
    }

    $supplies = Supply::active()
        ->where(function ($query) use ($q) {
            $query->where('name', 'like', "%{$q}%")
                  ->orWhere('code', 'like', "%{$q}%")
                  ->orWhere('brand', 'like', "%{$q}%");  // ← agregar
        })
        ->orderBy('name')
        ->limit(15)
        ->get(['id', 'code', 'name', 'brand', 'tracks_lot']);  // ← agregar brand

    return response()->json($supplies);
}
```

**Verificación:** `GET /supplies/search?q=labix` retorna insumos que tienen esa marca.
`GET /supplies/search?q=diluyen` sigue funcionando por nombre.

## PASO 3 — Actualizar `selectSupply()` en facturas de compra

En la función `supplySearch()` de `purchase-invoices/create.blade.php`,
modificar `selectSupply()` para que el texto del input y la descripción del ítem
incluyan la marca:

```javascript
selectSupply(supply) {
    // Armar el label completo: "Nombre — Marca" si tiene marca, solo "Nombre" si no
    const label = supply.brand
        ? `${supply.name} — ${supply.brand}`
        : supply.name;

    item.supply_id = supply.id;
    item.description = label;           // descripción del ítem incluye la marca
    item._supply_code = supply.code;
    item._supply_label = label;         // para el display del badge
    this.searchText = label;            // el input muestra "Nombre — Marca"
    this.showResults = false;

    this.$nextTick(() => {
        const qtyInput = document.querySelector(`#item-qty-${index}`);
        if (qtyInput) qtyInput.focus();
    });
},
```

## Notas / Consideraciones

- Los insumos sin marca (`brand = null` o `brand = ''`) no muestran el separador `—`,
  el label queda limpio como solo el nombre.
- La `description` del ítem en facturas de compra ahora incluye la marca — esto es
  intencional: si se guarda la factura, la descripción queda como `Nombre — Marca`.
  Si el usuario prefiere otra descripción puede editarla después de seleccionar.
- El badge simplificado (`código  nombre — marca  ×`) es más limpio que el badge anterior
  y usa menos espacio vertical en la tabla de ítems.
