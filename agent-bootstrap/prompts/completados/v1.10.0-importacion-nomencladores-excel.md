# v1.10.0 — Importación de nomencladores desde Excel

**VERSIÓN:** v1.10.0  
**SLUG:** importacion-nomencladores-excel  
**DEPENDENCIAS:** ninguna  
**ESTIMACIÓN:** 1-2h  
**PRIORIDAD:** alta

---

## Descripción

Los archivos `.xlsx` en `docs/` son **nomencladores base** — listas de prácticas con
sus valores NBU que sirven de referencia para facturación. NO son obras sociales.
Cada obra social (`Insurance`) elige qué nomenclador base usar a través del campo
`nomenclator_id` (dropdown "Nomenclador para Facturar" en la edición de la obra social).

Actualmente solo existe "NOMENCLADOR 20162023" como nomenclador base (visible en
`/nomenclator` → sección "Nomencladores Base"). Este seeder agrega 8 nomencladores
base más desde los archivos Excel.

Crear un seeder que lee los 8 archivos `.xlsx` de la carpeta `docs/`, crea cada uno
como `Insurance` con `type='nomenclador'`, y asocia cada práctica (`InsuranceTest`)
con su valor NBU. Si un código de test no existe en la tabla `tests`, se crea como
test nuevo.

Ajustar el seeder según lo que se encuentre.

## PASO 2 — Crear el seeder

Crear `database/seeders/NomencladoresExcelSeeder.php`.

### Estructura del seeder

```php
<?php

namespace Database\Seeders;

use Illuminate\Database\Seeder;
use App\Models\Test;
use App\Models\Insurance;
use App\Models\InsuranceTest;
use PhpOffice\PhpSpreadsheet\IOFactory;

class NomencladoresExcelSeeder extends Seeder
{
    /**
     * Mapeo de archivo → nombre del nomenclador
     */
    private array $nomencladores = [
        'pami.xlsx'       => 'PAMI',
        'medicus.xlsx'    => 'Medicus',
        'omint.xlsx'      => 'OMINT',
        'swiss2012.xlsx'  => 'Swiss Medical',
        'Issn2020.xlsx'   => 'ISSN',
        '2016.xlsx'       => 'Nomenclador 2016',
        '2016uni.xlsx'    => 'Nomenclador 2016 Uni',
        '2012reduc.xlsx'  => 'Nomenclador 2012 Reducido',
    ];

    public function run(): void
    {
        $docsPath = base_path('docs');

        foreach ($this->nomencladores as $filename => $nombre) {
            $filePath = $docsPath . DIRECTORY_SEPARATOR . $filename;

            if (!file_exists($filePath)) {
                $this->command->warn("Archivo no encontrado: {$filename}, saltando...");
                continue;
            }

            $this->command->info("Procesando: {$filename} → {$nombre}");
            $this->importNomenclador($filePath, $nombre);
        }

        $this->command->info('¡Importación de todos los nomencladores completada!');
    }

    private function importNomenclador(string $filePath, string $nombre): void
    {
        $spreadsheet = IOFactory::load($filePath);
        $sheet = $spreadsheet->getActiveSheet();
        $highestRow = $sheet->getHighestRow();

        // Crear/actualizar el Insurance con type=nomenclador
        $insurance = Insurance::firstOrCreate(
            ['name' => $nombre],
            [
                'type' => 'nomenclador',
                'tax_id' => null,
                'tax' => null,
                'phone' => null,
                'email' => null,
                'address' => null,
                'state' => null,
                'country' => 'argentina',
                'price' => 0,
                'nbu' => 0,
                'nbu_value' => 1,
                'group' => null,
                'instructions' => "Nomenclador importado desde {$nombre}",
            ]
        );

        if ($insurance->type !== 'nomenclador') {
            $insurance->update(['type' => 'nomenclador']);
        }

        $testsCreated = 0;
        $practicesCreated = 0;
        $practicesUpdated = 0;
        $skipped = 0;

        // Determinar si la primera fila es encabezado
        // AJUSTAR según lo encontrado en PASO 1
        $startRow = 2; // Si fila 1 es encabezado; cambiar a 1 si no hay encabezado

        // Columnas: AJUSTAR según lo encontrado en PASO 1
        $colCodigo = 'A';
        $colNombre = 'B';
        $colNbu    = 'C';

        for ($row = $startRow; $row <= $highestRow; $row++) {
            $codigo = trim((string) $sheet->getCell("{$colCodigo}{$row}")->getValue());
            $nombre_det = trim((string) $sheet->getCell("{$colNombre}{$row}")->getValue());
            $nbuRaw = $sheet->getCell("{$colNbu}{$row}")->getValue();

            // Saltar filas vacías
            if (empty($codigo) && empty($nombre_det)) {
                continue;
            }

            if (empty($codigo)) {
                $this->command->warn("  Fila {$row}: sin código, saltando");
                $skipped++;
                continue;
            }

            // Convertir NBU (manejar formato con coma o punto)
            $nbu = $this->parseNbu($nbuRaw);

            // Buscar o crear el test por código
            $test = Test::where('code', $codigo)->first();

            if (!$test) {
                $test = Test::create([
                    'code' => $codigo,
                    'name' => $nombre_det ?: "Test {$codigo}",
                    'nbu' => $nbu,
                    'unit' => null,
                    'parent' => null,
                    'low' => null,
                    'high' => null,
                    'instructions' => null,
                    'default_reference_category_id' => null,
                    'decimals' => null,
                    'negative' => null,
                    'positive' => null,
                    'questions' => null,
                    'method' => null,
                    'price' => null,
                    'cost' => null,
                    'work_sheet' => null,
                    'material' => null,
                    'formula' => null,
                    'box' => null,
                ]);
                $testsCreated++;
                $this->command->line("  Test creado: [{$codigo}] {$nombre_det}");
            }

            // Crear o actualizar InsuranceTest
            $insuranceTest = InsuranceTest::where('insurance_id', $insurance->id)
                ->where('test_id', $test->id)
                ->first();

            if ($insuranceTest) {
                $insuranceTest->update([
                    'nbu_units' => $nbu,
                ]);
                $practicesUpdated++;
            } else {
                InsuranceTest::create([
                    'insurance_id' => $insurance->id,
                    'test_id' => $test->id,
                    'nbu_units' => $nbu,
                    'price' => null,
                    'copago' => 0,
                    'requires_authorization' => false,
                ]);
                $practicesCreated++;
            }
        }

        $this->command->info("  Resultado {$nombre}:");
        $this->command->info("    Tests nuevos creados: {$testsCreated}");
        $this->command->info("    Prácticas creadas: {$practicesCreated}");
        $this->command->info("    Prácticas actualizadas: {$practicesUpdated}");
        if ($skipped > 0) {
            $this->command->warn("    Filas saltadas: {$skipped}");
        }
    }

    private function parseNbu($value): float
    {
        if (is_numeric($value)) {
            return (float) $value;
        }

        if (is_string($value)) {
            // Manejar formato europeo: "1.234,56" → 1234.56
            $clean = str_replace('.', '', $value); // quitar separador de miles
            $clean = str_replace(',', '.', $clean); // coma decimal → punto
            return (float) $clean;
        }

        return 0;
    }
}
```

### Decisiones de diseño

1. **Idempotente**: Usa `firstOrCreate` para Insurance y `updateOrCreate` lógica para
   InsuranceTest. Se puede correr múltiples veces.

2. **Tests nuevos**: Si un código no existe en `tests`, se crea un test nuevo con los
   datos mínimos (código, nombre, NBU). Igual que `Nomenclador20162023Seeder`.

3. **Formato NBU**: Maneja tanto formato numérico directo (Excel) como string con
   coma decimal (europeo).

4. **Logging**: Informa tests creados, prácticas creadas/actualizadas, filas saltadas.

## PASO 3 — Verificar y ajustar columnas

Después de crear el seeder, ejecutar SOLO la verificación del PASO 1 en tinker para
confirmar las columnas. Si algún archivo tiene estructura diferente (ej: código en
columna B en vez de A), agregar lógica de detección o mapeo por archivo.

Posibles variaciones:
- Algunos archivos pueden tener encabezado y otros no
- El orden de columnas puede variar entre archivos
- Algunos pueden tener columnas extra

Si hay variaciones significativas, agregar un mapeo por archivo:

```php
private array $columnMaps = [
    'pami.xlsx'      => ['codigo' => 'A', 'nombre' => 'B', 'nbu' => 'C', 'start' => 2],
    'medicus.xlsx'   => ['codigo' => 'A', 'nombre' => 'B', 'nbu' => 'C', 'start' => 2],
    // ... etc
];
```
