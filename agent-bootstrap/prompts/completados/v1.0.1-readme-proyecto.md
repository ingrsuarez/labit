## Instalación

1. Clonar el repositorio
2. composer install
3. npm install
4. Copiar .env.example a .env
5. php artisan key:generate
6. Configurar base de datos en .env
7. php artisan migrate
8. php artisan db:seed (si hay seeders)
9. npm run build (o npm run dev para desarrollo)
10. php artisan serve

## PASO 3 — Commit y tag

```bash

git add README.md
git commit --no-verify -m "docs: crear README profesional del proyecto

- Descripción general del sistema y módulos
- Stack tecnológico y requisitos
- Instrucciones de instalación paso a paso
- Roles del sistema y estructura del proyecto"

# Merge a develop
git checkout develop
git pull origin develop --ff-only
git merge feature/v1.0.1-readme-proyecto --no-ff
git push origin develop

# Merge a master
git checkout master
git pull origin master --ff-only
git merge develop --no-ff
git push origin master

# Tag
git tag -a v1.0.1 -m "Release v1.0.1: README del proyecto"
git push origin v1.0.1

# Limpieza
git branch -d feature/v1.0.1-readme-proyecto
```
