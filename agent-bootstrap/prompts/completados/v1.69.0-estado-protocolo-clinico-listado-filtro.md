# v1.69.0 — Estado del protocolo clínico: columna, filtro y actualización automática

**Tipo:** minor — nueva funcionalidad + corrección de comportamiento
**Prioridad:** alta — visibilidad operativa para bioquímicos y recepcionistas
**Dependencias:** ninguna (campo `status` y trait `Auditable` ya existen en `Admission`)

---

## Qué hace

Muestra el estado (Pendiente / En Proceso / Completado / Validado) en el listado de
admisiones del lab clínico (`/lab/admissions`), agrega un filtro por estado en la barra
de filtros, y corrige que el estado de la admisión se actualice automáticamente cuando se
cargan resultados o se validan/desvalidan prácticas.

Hoy el campo `status` existe en el modelo pero nunca se conectó al flujo de validación —
siempre queda en `pending` aunque haya prácticas completadas o validadas.

## PASO 2 — Agregar `getCalculatedStatusAttribute()` en `Admission`

Agregar en `app/Models/Admission.php` el mismo patrón que `VetAdmission`:

```php
/**
 * Estado calculado a partir del estado real de las prácticas.
 * Requiere que admissionTests esté cargado (eager load).
 */
public function getCalculatedStatusAttribute(): string
{
    // Excluir prácticas-padre agrupadoras (sin resultado propio, con hijos)
    // Si test.childTests no está cargado, incluir por defecto.
    $countable = $this->admissionTests->filter(function (AdmissionTest $at) {
        if ($at->hasResult() || $at->is_validated) {
            return true;
        }
        if ($at->relationLoaded('test') && $at->test?->relationLoaded('childTests')) {
            return $at->test->childTests->isEmpty();
        }
        return true;
    });

    $total = $countable->count();

    if ($total === 0) {
        return self::STATUS_PENDING;
    }

    $validated  = $countable->where('is_validated', true)->count();
    $withResult = $countable->filter(fn($at) => $at->hasResult())->count();

    if ($validated === $total) {
        return self::STATUS_VALIDATED;
    }

    if ($withResult === $total || $validated > 0) {
        return self::STATUS_COMPLETED;
    }

    if ($withResult > 0) {
        return self::STATUS_IN_PROGRESS;
    }

    return self::STATUS_PENDING;
}
```

Agregar también el import de `AdmissionTest` si no está en el modelo:
```php
// El modelo ya usa admissionTests via hasMany — no requiere import adicional
```
