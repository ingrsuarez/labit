## Descripción

Extiende el sistema de sedes (v1.30.0) para que cada usuario pueda configurar su sede de
laboratorio por defecto. Agrega un selector de sede en el header del lab para cambio al vuelo
(sesión), un helper `active_lab_branch_id()`, y filtro automático en los listados basado
en la sede activa del usuario.

## PASO 2 — Helper `active_lab_branch_id()`

Crear `app/Helpers/lab_branch.php`:

```php
<?php

use App\Models\LabBranch;

if (! function_exists('active_lab_branch_id')) {
    function active_lab_branch_id(): ?int
    {
        if (session()->has('active_lab_branch_id')) {
            return session('active_lab_branch_id') ?: null;
        }
        return auth()->user()?->default_lab_branch_id;
    }
}

if (! function_exists('active_lab_branch_name')) {
    function active_lab_branch_name(): string
    {
        $id = active_lab_branch_id();
        if (! $id) {
            return 'Todas las sedes';
        }
        return LabBranch::find($id)?->name ?? 'Todas las sedes';
    }
}
```

Registrar en `composer.json` → `autoload.files`:

```json
"files": [
    "app/Helpers/company.php",
    "app/Helpers/lab_branch.php"
]
```

Ejecutar `composer dump-autoload`.

**Verificación:** `php artisan tinker` → `active_lab_branch_id()` retorna null (sin sede en sesión ni default).

## PASO 3 — Rutas para cambio de sede

### 3.1 — Ruta `switch-lab-branch` (cambio en sesión)

Dentro del grupo `permission:lab.section`:

```php
Route::post('switch-lab-branch', function (Request $request) {
    $branchId = $request->lab_branch_id;
    if ($branchId) {
        $branch = \App\Models\LabBranch::findOrFail($branchId);
        session(['active_lab_branch_id' => $branch->id, 'active_lab_branch_name' => $branch->name]);
    } else {
        session()->forget(['active_lab_branch_id', 'active_lab_branch_name']);
    }
    return back();
})->name('switch-lab-branch');
```

### 3.2 — Ruta `user/default-branch` (guardar en perfil)

```php
Route::put('user/default-branch', function (Request $request) {
    $request->validate(['default_lab_branch_id' => 'nullable|exists:lab_branches,id']);
    auth()->user()->update(['default_lab_branch_id' => $request->default_lab_branch_id]);
    return back()->with('success', 'Sede por defecto actualizada.');
})->name('user.default-branch');
```

**Verificación:** Las rutas aparecen en `php artisan route:list`.

## Notas

- `default_lab_branch_id` es nullable. Usuarios sin sede ven todos los protocolos.
- La sede por defecto es una **preferencia**, no una restricción.
- El selector del header usa sesión para permitir cambio temporal sin modificar la preferencia.
- El helper sigue el mismo patrón que `active_company_id()` en `app/Helpers/company.php`.
